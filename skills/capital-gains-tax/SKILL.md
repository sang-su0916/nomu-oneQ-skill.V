# 양도소득세 계산기 (capital-gains-tax)

⚠️ **반드시 계산기 CLI를 실행하세요. 직접 계산 금지** — hallucination 방지.

⚠️ **부동산 기준 스킬**: 비상장주식·파생상품 양도세는 별개 체계이므로 본 스킬 사용 금지.

## 법령 근거

| 조문                 | 내용                                                   | 출처                        |
| -------------------- | ------------------------------------------------------ | --------------------------- |
| 소득세법 §55         | 기본세율 8단계 누진 (종소세와 동일)                    | law.go.kr (2026-04-14 조회) |
| 소득세법 §94         | 양도소득 과세대상·양도차익 산정                        | law.go.kr                   |
| 소득세법 §95         | 장기보유특별공제 (표1 일반 / 표2·표3 1세대1주택 특례)  | law.go.kr                   |
| 소득세법 §103        | 양도소득 기본공제 (연 250만원)                         | law.go.kr                   |
| 소득세법 §104        | 단기보유·중과세율 (1년 미만 70% 등 / 다주택 중과 유예) | law.go.kr                   |
| 소득세법 시행령 §154 | 1세대1주택 비과세 요건 — 12억원 한도 (2021.12.8 개정)  | law.go.kr                   |
| 지방세법 §92         | 지방소득세 = 산출세액의 10%                            | law.go.kr                   |

## 핵심 공식

```
양도차익 = 양도가액 - 취득가액 - 필요경비(자본적지출·양도비용)
과세 양도차익 = 양도차익  (1세대1주택 12억 초과 시 × (양도가액 - 12억) / 양도가액)
장특공 = 과세 양도차익 × 공제율
과세표준 = 과세 양도차익 - 장특공 - 기본공제(250만)
산출세액 = 과세표준 × 세율 - 누진공제   (단기보유 시 단일세율 × 과세표준)
지방소득세 = 산출세액 × 10%
```

## 세율표

### 기본세율 (소득세법 §55, 8단계 누진)

| 과세표준 구간                             | 세율 | 누진공제액 |
| ----------------------------------------- | :--: | ---------: |
| 14,000,000원 이하                         |  6%  |          0 |
| 14,000,000원 초과 ~ 50,000,000원 이하     | 15%  |  1,260,000 |
| 50,000,000원 초과 ~ 88,000,000원 이하     | 24%  |  5,760,000 |
| 88,000,000원 초과 ~ 150,000,000원 이하    | 35%  | 15,440,000 |
| 150,000,000원 초과 ~ 300,000,000원 이하   | 38%  | 19,940,000 |
| 300,000,000원 초과 ~ 500,000,000원 이하   | 40%  | 25,940,000 |
| 500,000,000원 초과 ~ 1,000,000,000원 이하 | 42%  | 35,940,000 |
| 1,000,000,000원 초과                      | 45%  | 65,940,000 |

### 단기보유 중과세율 (소득세법 §104)

| 자산 종류          | 1년 미만 | 1~2년 미만 | 2년 이상 |
| ------------------ | :------: | :--------: | :------: |
| 주택·입주권·분양권 |   70%    |    60%     | 기본세율 |
| 토지·기타          |   50%    |    40%     | 기본세율 |

### 장기보유특별공제 (소득세법 §95)

**일반 표1** (토지·비1주택): 3년 6% → 연 2%p → 15년 이상 30% (최대)

**1세대1주택 특례** (거주3년·보유3년 이상일 때만 적용):

- 표2 보유기간: 3년 12% → 연 4%p → 10년 이상 40%
- 표3 거주기간: 3년 12% → 연 4%p → 10년 이상 40%
- 합산 **최대 80%**

## CLI 사용법

### `calculate-gain` — 양도차익 산정

```bash
python3 skills/capital-gains-tax/references/calculator.py calculate-gain \
  --sale-price 1500000000 --acquisition-price 800000000 --necessary-expense 50000000
```

### `long-term-deduction` — 장기보유특별공제율·공제액

```bash
# 1세대1주택 특례 (보유10년·거주10년 → 80%)
python3 skills/capital-gains-tax/references/calculator.py long-term-deduction \
  --capital-gain 650000000 --holding-years 10 --is-one-house --residence-years 10
# 일반 (보유10년 → 16%)
python3 skills/capital-gains-tax/references/calculator.py long-term-deduction \
  --capital-gain 100000000 --holding-years 10
```

### `one-house-exemption` — 1세대1주택 비과세 판정

```bash
# 양도 11억·보유5년·거주2년·조정X → 전액 비과세
python3 skills/capital-gains-tax/references/calculator.py one-house-exemption \
  --sale-price 1100000000 --holding-years 5 --residence-years 2
# 양도 15억·차익 3억 → 12억 초과분 부분과세
python3 skills/capital-gains-tax/references/calculator.py one-house-exemption \
  --sale-price 1500000000 --holding-years 10 --capital-gain 300000000
```

### `basic-tax` — 기본세율 산출세액 (단기보유 중과 자동 적용)

```bash
# 기본세율
python3 skills/capital-gains-tax/references/calculator.py basic-tax --taxable-gain 127500000
# 주택 6개월 보유 → 70% 중과
python3 skills/capital-gains-tax/references/calculator.py basic-tax \
  --taxable-gain 100000000 --short-term-months 6 --asset-type house
```

### `full-calc` — 종합 계산

```bash
python3 skills/capital-gains-tax/references/calculator.py full-calc \
  --sale-price 1500000000 --acquisition-price 800000000 --necessary-expense 50000000 \
  --holding-years 10 --is-one-house --residence-years 10
```

양도차익 → 1세대1주택 비과세 판정 → 장특공 → 기본공제 → 산출세액 전 과정 JSON.

## 되묻기 규칙 (정보 부족 시)

1. **부동산 종류 불명**: "주택인가요, 토지인가요? 주택은 단기보유 중과율(1년 미만 70%)이 더 높고, 1세대1주택 특례 대상도 주택만 가능합니다."
2. **1주택/다주택 여부**: "1세대1주택 여부를 알려주세요. 다주택자는 비과세·장특공 80% 특례 대상이 아닙니다. (조정지역 중과는 현재 유예)"
3. **거주요건**: "2017.8.3 이후 조정대상지역 취득 주택은 거주 2년 이상 요건이 추가됩니다. 취득일·조정지역 여부 확인 필요."
4. **보유·거주 연수**: "장기보유특별공제와 1세대1주택 특례는 보유·거주 연수가 중요합니다. 등기부 기준 취득일·전입일 기준 실거주 기간을 알려주세요."
5. **필요경비**: "취득세·중개수수료·법무비·인테리어 자본적지출·양도시 중개수수료를 합산해주세요. (도배·장판 등 수익적 지출은 제외)"

## 도메인 특수 규칙

- **분류과세**: 양도소득세는 종합소득세와 별개로 과세 (합산 아님)
- **12억 초과 고가주택 부분과세**: 1세대1주택 요건 충족해도 양도가액 12억 초과분만큼만 과세
  - 과세 양도차익 = 양도차익 × (양도가액 - 12억) / 양도가액
- **다주택자 중과 유예**: 현재 정부 정책으로 조정지역 2주택 +20%p, 3주택 +30%p 중과가 유예 중 — **실제 신고 시 최신 정책 재확인 필수**
- **단기보유 우선**: 단기보유 중과세율과 기본세율이 모두 해당되면 **높은 세액 기준**으로 적용 (소득세법 §104⑤) — 본 계산기는 단기 플래그 입력 시 단기세율 적용
- **양도소득 기본공제 250만**: 1과세기간당 1회 (연 1회). `full-calc`에 자동 적용
- **장특공 3년 미만 0**: 1세대1주택 특례도 보유 3년 미만이면 표2·표3 모두 미적용 → 일반 표1도 0%

## 응답 포맷

계산 결과에 다음 항목을 포함하세요.

- 양도가액·취득가액·필요경비 (입력값 재확인)
- 양도차익 (산식 명시)
- 1세대1주택 비과세 판정 결과 (해당 시)
- 장기보유특별공제율·공제액 (표 구분: 일반/특례)
- 과세표준 (기본공제 250만 반영)
- 산출세액 + 지방소득세 + 총세부담
- 적용 세율 (구간 설명 / 단기 중과 여부)
- 법령 근거 (§55/§94/§95/§103/§104)
- 면책 문구

**면책**: 2026년 기준 양도소득세법. 조정대상지역·다주택자 중과(2주택 +20%p, 3주택 +30%p)는 현재 유예 상태 — 정부 정책에 따라 변동되므로 실제 신고 시 국세청·세무 전문가 확인 필수. 비사업용 토지·비거주자·감정가 기준 등 예외는 미반영. 지방소득세는 소득세의 10%로 단순 계산.

## 알려진 한계

1. **다주택 중과 미반영** — 조정지역 2주택 +20%p / 3주택 +30%p 중과는 현재 정책상 유예 중이라 본 계산기 미반영. 재시행 시 별도 계산 필요.
2. **비사업용 토지 중과 미반영** — §104의3 비사업용 토지 +10%p 중과 미반영.
3. **감정가·환산취득가액 미지원** — 취득가액은 입력 실거래가 기준. 1985년 이전 취득 등 환산취득가액 산정은 별도.
4. **양도소득공제 특례** — 상생임대주택·1세대1주택 특례·일시적 2주택·상속·혼인합가 등 특례 규정은 미반영.
5. **비거주자·법인** — 비거주자 양도세(§121), 법인의 토지·주택 양도 추가과세(법인세법 §55의2)는 별도.
6. **부동산 한정** — 비상장주식(§94①3)·파생상품·영업권 등은 본 계산기 범위 밖.
