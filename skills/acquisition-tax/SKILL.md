# 취득세 계산기 (acquisition-tax)

⚠️ **반드시 계산기 CLI를 실행하세요. 직접 계산 금지** — hallucination 방지.

⚠️ **부동산 기준 스킬**: 차량·회원권·건설기계 등 타 자산 취득세는 별개 세율 — 본 스킬 사용 금지.

## 법령 근거

| 조문               | 내용                                                                                       | 출처                        |
| ------------------ | ------------------------------------------------------------------------------------------ | --------------------------- |
| 지방세법 §11①1     | 상속 취득 — 농지 2.3% / 농지 외 2.8%                                                       | law.go.kr (2026-04-14 조회) |
| 지방세법 §11①2     | 무상취득(증여) — 3.5% (비영리 2.8%)                                                        | law.go.kr                   |
| 지방세법 §11①3,5,6 | 원시취득·공유분할 — 2.8% / 2.3%                                                            | law.go.kr                   |
| 지방세법 §11①7     | 유상 농지 3% / 농지 외 기타 4%                                                             | law.go.kr                   |
| 지방세법 §11①8     | 유상 주택 — 6억↓ 1% / 6~9억 공식세율 / 9억↑ 3%                                             | law.go.kr                   |
| 지방세법 §13의2①   | 다주택·법인 유상 주택 중과 — 1세대 2/3주택 조정 8%/12%, 3/4주택 비조정 8%/12%, 법인 12%    | law.go.kr                   |
| 지방세법 §13의2②   | 조정대상지역 증여 중과 — 시가표준 3억↑ 주택 12% (1세대1주택자 배우자·직계존비속 증여 제외) | law.go.kr                   |
| 농어촌특별세법 §5  | 취득세액의 10% (85㎡ 이하 국민주택 규모 면제)                                              | law.go.kr                   |
| 지방세법 §151      | 지방교육세 = 취득세율의 50% 수준 (주택 유상은 본세율의 10%)                                | law.go.kr                   |

## 핵심 공식

```
─ 유상 주택 표준세율 (§11①8) ────────────────────────
6억 이하                 : 1%
6억 초과 ~ 9억 이하      : (취득가액/1억 × 2/3 - 3) / 100  (소수점 4자리)
                           → 6억=1%, 7.5억=2%, 9억=3% 선형
9억 초과                 : 3%

─ 1세대 다주택 중과 (§13의2, 중과기준세율 2%) ────────
조정지역 1세대 2주택     : 4% + 2%×200% = 8%
조정지역 1세대 3주택↑    : 4% + 2%×400% = 12%
비조정 1세대 3주택       : 4% + 2%×200% = 8%
비조정 1세대 4주택↑      : 4% + 2%×400% = 12%

─ 법인 주택 유상 (§13의2) ────────────────────────────
                         : 4% + 2%×400% = 12%

─ 증여 무상 (§11①2, §13의2②) ────────────────────────
기본                     : 3.5% (비영리 2.8%)
조정+시가표준 3억↑ 주택  : 12% (1세대1주택자→배우자·직계 증여 제외)

─ 부가세 ──────────────────────────────────────────
농어촌특별세 = 취득세액 × 10%  (전용 85㎡ 이하 면제)
지방교육세    = 취득가액 × 본세율의 10% (주택 유상) 또는 근사 0.1~0.4%
총세부담 = 취득세 + 농특세 + 지방교육세
```

## 세율표

### 유상 주택 (§11①8)

| 취득가액 구간       |      세율      | 비고                     |
| ------------------- | :------------: | ------------------------ |
| 6억 이하            |       1%       | 실효세율 1.1% (85㎡↓)    |
| 6억 초과 ~ 9억 이하 | 1.0%~3.0% 선형 | 공식: (가액×2/3억-3)/100 |
| 9억 초과            |       3%       | 실효세율 3.3%            |

### 다주택·법인 중과 (§13의2)

| 구분                     | 세율 | 공식            |
| ------------------------ | :--: | --------------- |
| 조정 1세대 2주택         |  8%  | 4% + 2%×200%    |
| 조정 1세대 3주택↑        | 12%  | 4% + 2%×400%    |
| 비조정 1세대 3주택       |  8%  | 4% + 2%×200%    |
| 비조정 1세대 4주택↑      | 12%  | 4% + 2%×400%    |
| 법인 주택 유상           | 12%  | 4% + 2%×400%    |
| 조정+시가 3억↑ 주택 증여 | 12%  | 3.5% → 12% 중과 |

## 되묻기 규칙 (정보 부족 시)

1. **취득 유형 불명**: "유상 매매인가요, 증여인가요, 상속인가요? 각각 세율 체계가 다릅니다 (유상 1~3%, 증여 3.5%, 상속 2.3~2.8%)."
2. **주택 수 확인**: "취득 후 1세대가 보유하게 될 주택 수를 알려주세요. 조정지역은 2주택부터, 비조정지역은 3주택부터 중과 대상입니다."
3. **조정대상지역 여부**: "취득 주택이 조정대상지역 소재인가요? 2026-04 현재 서울 강남·서초·송파·용산 4구만 조정지역입니다 — 국토부 공고 확인 필요."
4. **전용면적**: "전용면적 알려주세요. 85㎡ 이하 국민주택 규모는 농어촌특별세가 면제됩니다."
5. **법인/개인**: "법인 명의로 취득하나요? 법인 주택 유상 취득은 12% 중과 대상입니다 (대도시 법인 중과 예외 별도)."
6. **증여자 관계**: "증여자가 1세대1주택자인가요? 1주택자가 배우자·직계존비속에게 증여 시 조정지역 12% 중과에서 제외됩니다."

## 도메인 특수 규칙

- **6~9억 공식세율**: `rate = (취득가액/1억 × 2/3 - 3) / 100`, 소수점 넷째 자리 반올림. 6억 정확히 1%, 9억 정확히 3%로 수렴.
- **일시적 2주택 제외**: 이사·상속 등 일시적 2주택은 중과 제외 (시행령 §28의5) — 현재 계산기는 "취득 후 최종 주택 수" 기준으로 단순 처리하므로 실무에서는 별도 확인.
- **중과기준세율 2%**: §13의2 중과는 "표준 4% + 중과기준 2% × 가산%" 구조. 200% → 8%, 400% → 12%.
- **조정지역 증여 중과 예외**: 1세대1주택자가 배우자·직계존비속에게 증여 시 `--is-spouse-or-lineal-from-one-house` 플래그로 중과 제외 처리.
- **농특세 85㎡ 면제**: 국민주택 규모(전용 85㎡ 이하) 주택은 농어촌특별세법 §4⑪로 면제.
- **지방교육세 근사**: 주택 유상은 본세율 × 10% (1%→0.1%, 3%→0.3%), 그 외 `(본세율-2%)×50%` 근사식. 실무 신고 시 최종 수치는 지자체 고지서 기준.

## 응답 포맷

계산 결과에 다음 항목을 포함하세요.

- 취득가액·취득 유형·주택 수·조정지역 여부 (입력값 재확인)
- 적용 세율 + 적용 구간 설명 (`applied_rate_label`)
- 취득세 본세
- 농어촌특별세 (85㎡ 기준 면제 여부 명시)
- 지방교육세
- 총세부담 (`total_tax`)
- 법령 근거 (§11 / §13의2)
- 면책 문구

**면책**: 2026-04 기준 지방세법. 조정대상지역 지정 여부는 국토부 공고 별도 확인 필요 (2026-04 현재 서울 강남·서초·송파·용산 4구만 지정). 생애최초 감면·일시적 2주택·대도시 법인 중과 예외·상속 1주택 특례 등 개별 감면·특례는 미반영. 실제 신고 시 지방자치단체·세무 전문가 확인 필수.

## 알려진 한계

1. **조정대상지역 공고 별도 확인** — 2023.1 서울 4구(강남·서초·송파·용산) 외 전 지역 해제. 향후 지정 변동 시 사용자가 `--in-adjusted-area` 플래그로 입력.
2. **대도시 법인 중과 예외 미반영** — 지방세법 §13 대도시 내 법인 설립·전입 5년 내 부동산 취득 중과(3배)는 미반영. 본 스킬은 §13의2 주택 중과(12%)만 다룸.
3. **개별 감면 미반영** — 생애최초 주택 구입 감면(§36의5), 출산·양육 감면, 장애인 감면, 상속 1주택 감면, 일시적 2주택 예외(시행령 §28의5) 등 미반영.
4. **시가표준액 vs 시가인정액** — 증여는 원칙상 시가표준액, 특수관계자간은 시가인정액 적용 가능 (§10의2). 본 스킬은 입력값을 단일 과세표준으로 처리.
5. **환산취득가액 미지원** — 상속·증여·경매 등 비일반 취득가액 산정은 별도.
6. **자동차·회원권·건설기계** — 본 계산기는 부동산 한정.
