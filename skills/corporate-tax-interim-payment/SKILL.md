# 법인세 중간예납 계산기 (corporate-tax-interim-payment)

⚠️ **반드시 계산기 CLI를 실행하세요. 직접 계산 금지** — hallucination 방지.

## 언제 사용

- 12월 결산 법인 중간예납 기한(8월 31일) 전 납부세액 산출
- 직전사업연도 기준방식(방법1)과 가결산 방식(방법2) 중 유리한 방식 선택
- 기중 실적이 크게 악화된 경우 가결산 방식으로 절세 가능성 검토
- 신설법인·직전 산출세액 없는 경우 면제 해당 여부 판정

## 법인세율 (법인세법 §55 — 2026년 개정)

| 과세표준 구간             | 세율 | 누진공제     |
| ------------------------- | ---- | ------------ |
| 2억 이하                  | 10%  | 0            |
| 2억 초과 ~ 200억 이하     | 20%  | 2,000만      |
| 200억 초과 ~ 3,000억 이하 | 22%  | 4억 2,000만  |
| 3,000억 초과              | 25%  | 94억 2,000만 |

## 핵심 공식

```
─ 방법1 (직전사업연도 기준방식) ───────────────────────────────
중간예납세액 = (직전산출세액 - 공제·감면 - 원천징수 - 수시부과)
              × (중간예납기간 월수 / 직전사업연도 월수)

─ 방법2 (가결산 방식) ──────────────────────────────────────────
연환산 과세표준 = 중간예납기간 과세표준 × (12 / 중간예납월수)
연환산 산출세액 = 법인세 누진세율 적용 (§55)
중간예납 산출세액 = 연환산 세액 × (중간예납월수 / 12)
납부세액 = 중간예납 산출세액 - 공제·감면 - 원천징수

─ 납부기한 ────────────────────────────────────────────────────
중간예납기간 종료일로부터 2개월 이내
12월 결산법인 기준: 8월 31일
```

## CLI 사용법

### `standard` — 방법1 (직전사업연도 기준방식)

```bash
python3 skills/corporate-tax-interim-payment/references/calculator.py standard \
  --prior-tax 50000000 \
  --prior-months 12 \
  --current-period-months 6
```

기타 플래그: `--prior-credits` (공제·감면), `--prior-withholding` (원천징수), `--prior-occasional-tax` (수시부과).

### `estimation` — 방법2 (가결산 방식)

```bash
python3 skills/corporate-tax-interim-payment/references/calculator.py estimation \
  --interim-taxable-income 150000000 \
  --current-period-months 6
```

기타 플래그: `--credits` (공제·감면), `--withholding` (원천징수).

### `compare` — 두 방식 비교

```bash
python3 skills/corporate-tax-interim-payment/references/calculator.py compare \
  --prior-tax 100000000 \
  --interim-taxable-income 120000000 \
  --prior-months 12 \
  --current-period-months 6
```

출력: 방법1·방법2 각 세액 + `savings`(절감액) + `recommended`(권장) + `rationale`.

### `exemption-check` — 면제 대상 판정

```bash
python3 skills/corporate-tax-interim-payment/references/calculator.py exemption-check \
  --prior-tax 0
```

플래그:

- `--is-newly-incorporated` (신설법인 flag)
- `--prior-tax-finalized false` (직전 세액 미확정)

## 되묻기 규칙 (정보 부족 시)

1. **직전사업연도 월수**: "직전 사업연도가 12개월인가요? 합병·신설·사업연도 변경으로 단기 사업연도인 경우 월수 입력 필요합니다."
2. **공제·감면 포함 여부**: "직전 사업연도에 세액공제·감면 있었나요? (연구개발비 세액공제·중소기업 특별세액감면 등)"
3. **가결산 과세표준 확정 여부**: "상반기 과세표준이 가결산 기준인가요, 세무조정 후 확정치인가요? 가결산은 세무조정 전이라 실제 신고와 차이 발생 가능합니다."
4. **신설법인 요건**: "설립 후 최초 사업연도인가요? 분할신설법인·합병신설법인도 포함됩니다."

## 도메인 특수 규칙

- **방법 선택 자유**: 납세자는 방법1·방법2 중 유리한 쪽 자유 선택 가능. `compare` 서브커맨드 활용 권장.
- **가결산 추가 비용**: 방법2는 중간 결산 작성 비용·노력이 발생. 절감액이 크지 않으면 방법1이 현실적.
- **지방법인세 별도**: 본 계산기는 법인세(국세)만 산출. 지방법인세(법인세의 10%)는 별도 납부.
- **가산세 미반영**: 납부 지연 시 납부지연가산세(1일 0.022%) 발생. 본 계산기 미포함.
- **preliminary-closing 시너지**: 방법2(가결산) 진입 전 `preliminary-closing` 스킬로 상반기 손익 추정 → 과세표준 산출 후 본 스킬 `estimation` 적용.

## 알려진 한계

1. **지방법인세 미포함** — 법인세 × 10%. 실제 납부 총액은 법인세 + 지방법인세.
2. **세무조정 미반영** — 방법2 가결산 과세표준은 세무조정(손금불산입·익금산입) 전 추정치. 실제 신고 시 차이 가능.
3. **가산세 미계산** — 납부 지연·신고 불성실 가산세 미포함.
4. **연결납세 미지원** — 연결법인 중간예납은 별도 규정 적용.
5. **사업연도 변경법인** — 중간예납기간이 6개월이 아닌 경우 `--current-period-months` 수동 입력 필요.

## 관련 스킬

- `preliminary-closing` — 상반기 가결산 추정 → 방법2 과세표준 산출 연계 활용
- `income-tax` — 개인 종합소득세 (법인세는 본 스킬)
- `depreciation` — 결산조정 감가상각비 계산 (가결산 방식 과세표준 산출 시 활용)
- `financial-ratio` — 확정 결산 수치 기반 재무비율
- `omsc` — 본 스킬은 OMSC scaffold 기반 세무 도메인 스킬
