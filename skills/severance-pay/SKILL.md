# 퇴직금 계산기 (severance-pay)

⚠️ **반드시 계산기 CLI를 실행하세요. 직접 계산 금지** — hallucination 방지.

## 법령 근거

- **근로자퇴직급여보장법 §8** — 퇴직금 산정
- **근로기준법 §2(1)6** — 평균임금 정의
- 1년 이상 계속근로한 근로자가 퇴직 시 청구권 발생 (1년 미만 → 청구권 없음)
- 4주간 1주 평균 15시간 미만 단시간 근로자 → 적용 제외

## 핵심 공식

```
1일 평균임금 = (직전 3개월 임금총액 + 상여금×3/12 + 연차수당×3/12) ÷ 3개월 일수
퇴직금 = 1일 평균임금 × 30일 × (총 재직일수 / 365)
```

**중간정산 공제**: 과거 중간정산을 받은 경우 그 시점 이후 재직기간만 계산하거나, 전체 산정액에서 중간정산액 차감.

## 두 가지 모드

### 1. 정확 모드 (`calculate`)
직전 3개월 임금 총액 + 일수 + 상여/연차 + 총재직일수 입력 → 법정 산정.

```bash
python3 skills/severance-pay/references/calculator.py calculate \
  --avg-3month-wage-total 10500000 \
  --days-in-3month 92 \
  --annual-bonus 4000000 \
  --annual-leave-pay 1500000 \
  --total-service-days 1825 \
  --prior-settlement 0
```

### 2. 간이 모드 (`simple`)
평균월급 + 재직년수만으로 빠른 추정. **정확도 낮음** — 상담 초기 안내용.

```bash
python3 skills/severance-pay/references/calculator.py simple \
  --avg-monthly-wage 3500000 \
  --years 5 \
  --months 3
```

## 입력 파싱 가이드

사용자가 정보를 일부만 줘도 우선 `simple` 모드로 추정 후, 정확한 계산 필요 시 추가 정보 요청.

| 사용자 입력 | 모드 | 누락 시 기본값 |
|------------|------|---------------|
| "월급 350만, 5년 3개월" | simple | months=0 |
| "직전 3개월 1050만, 89일, 5년" | calculate | bonus=0, leave_pay=0 |
| "퇴직금 얼마? 5년 다녔어" | 되묻기 1번 | 평균월급 또는 직전 3개월 임금 |

## 되묻기 규칙 (정보 부족 시)

1. **임금 정보 부족**: "직전 3개월 임금 총액(또는 평균월급)을 알려주세요."
2. **재직기간 부족**: "입사일과 퇴직일(또는 총 재직년수·월수)을 알려주세요."
3. **상여/연차 별도 지급**: "정기 상여금이나 연차수당을 별도로 받으셨나요? (선택)"
4. **중간정산 여부**: "과거 퇴직금 중간정산을 받으신 적 있으신가요?"

## 응답 포맷

계산 결과 + 다음 항목 안내:
- 1일 평균임금
- 총 재직일수 / 재직년수
- 총 산정액
- 중간정산 차감 후 최종 지급액
- 산식 (검증용)
- 면책: "단체협약·취업규칙·DC형 가입 여부에 따라 실제 금액이 달라질 수 있습니다."

## 알려진 한계

1. **DC형(확정기여) 미지원** — DC형은 사업주가 매월 1/12 이상 적립한 금액 + 운용수익. 별도 가입사 명세서 확인 필요.
2. **임금피크제·휴직기간 보정 없음** — 평균임금 산정 시 정직·휴직·육아휴직 기간 제외 규정 미반영.
3. **세금 미산정** — 퇴직소득세 별도 계산 (퇴직소득공제 + 환산급여 + 세율표).
4. **상여금 산정 방식** — 본 계산은 "연간 상여금 ÷ 12 × 3" 단순화. 실제는 변동상여 평균 등 변형 존재.

## 관련 스킬

- `wage-base` — 평균임금/통상임금 산정 베이스 (상여금·연차수당 포함 범위)
- `four-insurances` — 퇴직금 지급 시 4대보험 정산
- `unemployment-benefit` — 퇴직 후 실업급여 자격
