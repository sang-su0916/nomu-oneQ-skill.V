# 감가상각 계산기 (depreciation)

⚠️ **반드시 계산기 CLI를 실행하세요. 직접 계산 금지** — hallucination 방지.

## 법령 근거

| 조문                  | 내용                                                   | 출처                                                  |
| --------------------- | ------------------------------------------------------ | ----------------------------------------------------- |
| 법인세법 §23          | 감가상각비의 손금산입 (결산조정·신고조정 한도 범위 내) | [국세청 nts.go.kr](https://www.nts.go.kr) / law.go.kr |
| 법인세법 시행령 §26   | 감가상각방법 (정액법·정률법·생산량비례법 등)           | law.go.kr                                             |

| 일반기업회계기준 §10  | 유형자산 (회계상 상각방법·잔존가치·내용연수 추정)      | 한국회계기준원 kasb.or.kr                             |

## 핵심 공식

```
정액법       = (취득가액 - 잔존가치) / 내용연수
정률법       = 기초 장부가액 × 상각률
  상각률     = 1 - (0.05)^(1/내용연수)    # 잔존가치율 5% (국내 관행)
생산량비례법 = (취득가액 - 잔존가치) / 총추정생산량 × 실제생산량
```

**세무 vs 회계**: 세법은 방법·내용연수 신고 필요. 상각범위액을 초과한 비용은 `감가상각비 한도초과액`으로 손금불산입(세무조정) 대상.

## 기준내용연수 대표값 (시행규칙 별표5·6)

| 카테고리                 | 자산            | 기준내용연수 |
| ------------------------ | --------------- | ------------ |
| `building-steel`         | 철골·철근 건물  | 40년         |
| `building-brick`         | 벽돌·석조 건물  | 20년         |
| `building-wood`          | 목조 건물       | 15년         |
| `vehicle`                | 차량운반구      | 5년          |
| `machinery-general`      | 일반 기계장치   | 8년          |
| `machinery-heavy`        | 중기계장치      | 10년         |
| `office-equipment`       | 사무용 비품     | 5년          |
| `computer`               | 컴퓨터·주변기기 | 4년          |
| `furniture`              | 집기·가구       | 5년          |
| `leasehold-improvements` | 임차자산 개량권 | 5년          |

**주의**: 위 값은 대표 참고값. 실제 업종·자산 세부분류는 시행규칙 별표5·6 원문 확인 필요 (±25% 범위에서 신고가능기간 존재).

## CLI 사용법

### `straight-line` — 정액법

```bash
python3 skills/depreciation/references/calculator.py straight-line \
  --acquisition-cost 10000000 --useful-life 5
# annual_depreciation = 2,000,000
```

옵션: `--salvage-value` (잔존가치, 기본 0), `--year` (특정 연도만 출력)

### `declining-balance` — 정률법

```bash
python3 skills/depreciation/references/calculator.py declining-balance \
  --acquisition-cost 10000000 --useful-life 5
```

옵션: `--rate` (수동 지정, 미지정 시 1 - 0.05^(1/n) 자동산출), `--year`

### `production` — 생산량비례법

```bash
python3 skills/depreciation/references/calculator.py production \
  --acquisition-cost 10000000 --salvage-value 0 \
  --total-production 100000 --actual-production 20000
# current_depreciation = 2,000,000
```

### `estimate-useful-life` — 기준내용연수 조회

```bash
python3 skills/depreciation/references/calculator.py estimate-useful-life \
  --asset-category computer
# useful_life_years = 4
```

## 입력 파싱 가이드

| 사용자 입력 예            | 액션                     | 비고                       |
| ------------------------- | ------------------------ | -------------------------- |
| "1천만 컴퓨터 4년 정액법" | `straight-line` 즉시     | 잔존 0 가정                |
| "기계 1억 정률법 8년"     | `declining-balance` 즉시 | 상각률 자동산출            |
| "차량 내용연수 몇 년?"    | `estimate-useful-life`   | `--asset-category vehicle` |
| "생산량 기준 기계 상각"   | `production`             | 총생산량·실제생산량 질의   |

## 되묻기 규칙

1. **상각방법 미지정**: "정액법/정률법/생산량비례법 중 어느 것인가요? (일반적으로 건물은 정액법, 기계는 정률법)"
2. **잔존가치**: "잔존가치는 보통 0으로 처리합니다. 별도 값이 있으면 알려주세요."
3. **내용연수**: "자산 종류를 알려주시면 `estimate-useful-life`로 시행규칙 별표5 기준 기본값을 조회할 수 있습니다."
4. **기준시점**: "취득연도와 현재 경과연수도 필요합니다 (월할상각 고려 시 취득일)."

## 도메인 특수 규칙

- **월할상각**: 사업연도 중 취득·처분 시 월할 계산 (본 계산기는 연단위만 지원)
- **잔존가치율 5%**: 국내 법인세법 관행상 정률법 계산 시 최종 잔존가치 5% 가정
- **상각방법 신고**: 세법상 법인이 최초 사업연도 과세표준 신고 시 상각방법 선택, 이후 변경 시 승인 필요
- **감가상각의제**: 이월결손금 등 이유로 상각비 미계상 시 세법상 감가상각의제 적용 (본 계산기 미지원)
- **즉시상각 특례**: 100만원 이하 소액자산·단기사용자산은 즉시 손금산입 가능 (법인세법 시행령 §31)

## 응답 포맷

- 취득가액·잔존가치·내용연수 (입력값 재확인)
- 상각방법 및 공식
- 연간 상각액 + 누적 상각 + 장부가액 (연도별 테이블)
- 상각률 (정률법, 소수점 4자리)
- 면책 문구

**면책**: 본 계산은 일반기업회계기준 및 법인세법 시행령 §26 계산식 기준 이론값입니다. 실무 적용 시 시행규칙 별표5·6 업종별 기준내용연수, 회사 회계정책(잔존가치 추정), 월할상각, 상각범위액 초과분 세무조정을 반드시 확인하세요.

## 알려진 한계

1. **월할상각 미지원** — 사업연도 중 취득·처분 시 월수 안분 로직 없음. 연단위로만 계산.
2. **세무조정 미반영** — 상각범위액 초과 손금불산입 (감가상각비한도초과액) 자동 판정 없음. 결산조정 금액과의 비교는 별도 필요.
3. **감가상각의제 미지원** — 이월결손금 등으로 상각 누락 시 세법상 의제 상각 규정 미반영.
4. **자본적지출·수익적지출 구분 없음** — 자본화 대상 판단은 사전 결정 후 금액만 입력.
5. **즉시상각 특례 미적용** — 100만원 이하 소액자산, 6개월 이내 단기자산 특례 자동판정 없음.
6. **업종별 내용연수 세분화 미지원** — `estimate-useful-life`는 대표값만 제공. 제조업 세부 업종별 기준내용연수(별표5·6)는 시행규칙 원문 확인 필요.
7. **정률법 최종연도 보정 없음** — 이론적으로 장부가액이 잔존가치(취득가액 × 5%)에 도달하면 상각 중단하나, 본 계산기는 내용연수 전 기간 기계적으로 적용.
