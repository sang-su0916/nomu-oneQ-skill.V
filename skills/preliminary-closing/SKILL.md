# 가결산 추정기 (preliminary-closing)

⚠️ **반드시 계산기 CLI를 실행하세요. 직접 계산 금지** — hallucination 방지.

## 언제 사용

- 월말·분기말 마감 전 당월·누적 손익을 빠르게 추정할 때
- 기중 YTD 수치를 연환산해 연말 손익을 선행 예측할 때
- 연간 목표(매출·영업이익·당기순이익) 대비 진도율을 점검할 때
- 법인세 중간예납(가결산 방식) 과세표준·추정세액 확인 전 기초 자료
- 결산 전 미지급·선급·감가·충당금 등 결산조정 누락 여부 체크리스트 확보

## 핵심 공식

```
─ 손익 구조 ────────────────────────────────────────────────
매출총이익   = 매출 − 매출원가
영업이익     = 매출총이익 − 판관비
세전이익     = 영업이익 + (영업외수익 − 영업외비용) − 이자비용
추정법인세   = max(세전이익, 0) × 예상 세율
당기순이익   = 세전이익 − 추정법인세

─ 마진 ────────────────────────────────────────────────────
매출총이익률 = 매출총이익 / 매출 × 100 (%)
영업이익률   = 영업이익 / 매출 × 100 (%)
순이익률     = 당기순이익 / 매출 × 100 (%)

─ 연환산 (YTD 단순선형) ──────────────────────────────────
연환산 금액  = YTD × (12 / 경과월수)
            ※ 계절성·일회성 손익 미반영 — 참고치

─ 목표 달성률 ────────────────────────────────────────────
달성률(%)    = 실적 / 목표 × 100
진도율(%)    = 경과월수 / 12 × 100
플래그 기준  = 달성률 − 진도율이 ±5%p 범위 이탈 시 "저조" / "상회"
```

## CLI 사용법

### `monthly` — 단월 가결산

```bash
python3 skills/preliminary-closing/references/calculator.py monthly \
  --revenue 100000000 \
  --cogs 60000000 \
  --sga 25000000 \
  --interest-expense 2000000
```

기타 플래그: `--other-income`, `--other-expense`, `--estimated-tax-rate` (default 0.20).

### `ytd` — 누적 가결산 + 연환산

```bash
python3 skills/preliminary-closing/references/calculator.py ytd \
  --ytd-revenue 600000000 \
  --ytd-cogs 360000000 \
  --ytd-sga 150000000 \
  --ytd-interest-expense 12000000 \
  --months-elapsed 6
```

출력: `ytd`(누적 손익) + `annualized`(연환산) 블록.

### `target-vs-actual` — 목표 대비 실적 진도율

```bash
python3 skills/preliminary-closing/references/calculator.py target-vs-actual \
  --target-revenue 1200000000 --actual-revenue 550000000 \
  --target-operating-income 120000000 --actual-operating-income 70000000 \
  --target-net-income 90000000 --actual-net-income 50000000 \
  --months-elapsed 6
```

출력: 항목별 달성률 + 진도율 대비 ±5%p 벗어나면 `flag` 표시 + 종합 상태 문장.

### `adjustment-checklist` — 결산조정 체크리스트

```bash
python3 skills/preliminary-closing/references/calculator.py adjustment-checklist
```

옵션: `--month` (표시용 라벨, 예: `2026-04`).

출력: 카테고리별 체크 항목(JSON) — 매출/채권·재고·고정자산·선급/미지급·세금·충당금 6개 카테고리.

| 사용자 입력 예                                 | 액션                   | 비고                         |
| ---------------------------------------------- | ---------------------- | ---------------------------- |

## 되묻기 규칙 (정보 부족 시)

1. **매출·원가 인식 기준**: "세금계산서 발행 기준인가요, 발생주의(서비스 제공·재화 인도) 기준인가요? 월말 집계에 차이가 날 수 있습니다."
2. **판관비 정의**: "인건비·임차료·감가상각비·광고선전비 포함 금액인가요? 미지급분(급여·이자) 포함 여부도 확인 필요합니다."
3. **예상 세율**: "법인세 세율(과표 2억 이하 9% + 지방세 0.9% = 약 9.9%, 2억 초과 19% + 지방세 1.9% = 약 20.9%) 중 어떤 구간인가요? default 0.20은 지방세 포함 과표 2억 초과 구간 반영치입니다."
4. **연환산 계절성**: "YTD × 12/경과월은 선형 가정입니다. 성수기·비수기 편차가 크다면 연환산 수치는 참고로만 활용하세요."
5. **목표 수치 기준**: "연초 수립 목표인가요, 수정 예산인가요? 변동한 목표 대비 진도율은 달라집니다."

## 도메인 특수 규칙

- **가결산 ≠ 결산 확정치**: 월말 마감 전 선급·미지급·감가·충당금 등 결산조정이 반영되지 않은 수치. 확정 손익과 차이 발생 가능.
- **법인세 추정치 한계**: 가결산 법인세는 세무조정(손금불산입·익금산입) 전 수치. 실제 법인세 신고 시 금액 달라짐.
- **연환산 선형 가정**: 월별 계절성, 일회성 손익(자산매각·대손상각), 연말 귀속 상여 등 고려 불가. 반드시 "참고치" 명시.
- **중간예납 선택권**: 법인세법 §63은 직전 사업연도 기준 납부(기본)와 가결산 기준 납부(선택) 2가지 방식을 허용. 기중 실적이 크게 악화된 경우 가결산 방식 유리.
- **플래그 ±5%p 임계**: 진도율 대비 ±5%p는 편의상 기준치. 업종·경영진 목표 난이도에 따라 조정 필요.

## 응답 포맷

계산 결과에 다음 항목을 포함하세요.

- 입력 수치 요약
- 손익 단계별 수치 (총이익·영업이익·세전이익·당기순이익)
- 마진율 3종 (매출총이익률·영업이익률·순이익률)
- ytd 모드 시: 연환산 블록
- target-vs-actual 모드 시: 항목별 달성률·flag·종합 상태
- adjustment-checklist 모드 시: 카테고리별 체크 항목 + 총 항목 수
- 면책 문구

**면책**: 가결산은 확정치 아님. 월말 마감 전 선급·미지급·감가 등 결산조정 미반영 가능성. 법인세는 예상 세율 기준 추정치로 실제 신고 시 차이.

## 알려진 한계

1. **결산조정 미반영** — 선급비용 기간배분, 미지급비용 인식, 감가상각비 월할 계상, 충당금 전입 등 결산조정 항목은 입력에 포함되어야 반영. 자동 추정 없음.
2. **세무조정 미반영** — 가결산 법인세는 회계상 세전이익 × 세율 단순 추정. 손금불산입(접대비 한도초과·기부금 한도초과), 익금산입(임대보증금 간주임대료) 등 세무조정 없음.
3. **연환산 선형 가정** — 계절성 강한 업종(유통·관광·교육)은 연환산 수치 오차 큼. 월별 가중치·전년 동기 배분 필요 시 수동 보정.
4. **목표 플래그 고정 임계** — 진도율 대비 ±5%p는 고정. 보수적 목표(낮은 기준)와 도전적 목표(높은 기준)에 동일 임계 적용은 부적절할 수 있음.
5. **현금흐름 미포함** — 손익 추정만 제공. 매출채권·재고·지급 시점 차이에 따른 현금흐름 영향은 `cash-flow-analysis` 스킬 별도 활용.
6. **중간예납 세액 직접 계산 미지원** — 본 스킬은 가결산 세전이익 기준 추정세액만 제공. 실제 중간예납 신고는 세무사 검토 필수.

## 관련 스킬

- `financial-ratio` — 확정 결산 수치로 재무비율 분석
- `financial-diagnosis` — 확정 결산 수치 기반 종합 재무 진단 (A~D 등급)
- `financial-statement-trend` — 수평·수직·추세 분석 (YoY, 공통형, CAGR)
- `cash-flow-analysis` — 영업·투자·재무활동 현금흐름 분석
- `break-even` — 손익분기점·목표이익 수량 산출
- `depreciation` — 결산조정 중 감가상각비 계상 지원
- `income-tax` — 개인 종합소득세 (법인세는 본 스킬의 추정세액 참고)
- `omsc` — 본 스킬은 OMSC scaffold로 생성된 경영 도메인 스킬
