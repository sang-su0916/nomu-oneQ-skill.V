# 주주총회 의사록 (shareholder-meeting-minutes)

> ⚠️ **본 체크리스트는 1차 작성 보조 도구입니다.** 실제 의사록은 **변호사·법무사 자문**을 거쳐 확정하고, 등기사항(임원 변경·정관변경·자본금 변경 등)은 **공증·등기**가 별도로 필요합니다.

비상장 중소기업 대표(상수님 타깃) 기준, 상법 규정에 따른 **정기·임시 주주총회 의사록**을 작성·검토하는 보조 도구입니다. 소집통지 마감, 의결 정족수, 필수 기재사항, 안건 문구를 하나로 통합했습니다.

## 법적 근거 (상법)

| 조문          | 내용                                                                                                                 |
| ------------- | -------------------------------------------------------------------------------------------------------------------- |
| **§363**      | 소집통지 — 주총 2주 전 서면/전자 발송. 자본금 10억↓ 소규모회사는 10일 전 가능(§363⑤). 주주 전원동의 시 소집절차 생략 |
| **§363의2**   | 주주제안권 — 일정 요건 주주의 의안 제안권                                                                            |
| **§366**      | 소수주주 총회소집청구권                                                                                              |
| **§368**      | **보통결의 — 출석 의결권 과반수 + 발행주식총수 1/4 이상 찬성**                                                       |
| **§368의3**   | 서면에 의한 의결권 행사 (정관 근거)                                                                                  |
| **§368의4**   | 전자적 방법에 의한 의결권 행사 (정관 근거) ⚠️ 서면투표와 구분                                                        |
| **§369**      | 1주 1의결권 원칙, 자기주식·상호주 의결권 제한                                                                        |
| **§373**      | **의사록 필수 기재 — ① 의사의 경과요령 ② 결과 ③ 의장 및 출석이사 기명날인 또는 서명**                                |
| **§374**      | 특별결의 대상 — 영업 전부/중요 일부 양도·양수, 영업임대, 경영위임, 타회사 영업 전부 양수, 중요재산 처분, 합병        |
| **§383①단서** | 소규모회사 특례 — 자본금 10억↓ → 이사 1명 가능(이사회 불요)                                                          |
| **§434**      | **특별결의 정족수 — 출석 의결권 2/3 이상 + 발행주식총수 1/3 이상 찬성**                                              |

## 결의 유형 (보통결의 vs 특별결의)

> ⚠️ 실무 용어로는 **보통결의**와 **특별결의** 2종만 사용합니다("특수결의"라는 표현은 지양).

| 유형                | 정족수                                                            | 대표 안건                                                                                                   |
| ------------------- | ----------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------- |
| **보통결의** (§368) | **발행주식총수 1/4 이상** + **출석 의결권 과반수(1/2 초과)** 찬성 | 재무제표 승인, 이익배당, 이사·감사 선임·해임(일반), 임원 보수한도, 주식매수선택권 부여 승인                 |
| **특별결의** (§434) | **발행주식총수 1/3 이상** + **출석 의결권 2/3 이상** 찬성         | 정관변경, 합병·분할, 영업양도(§374), 주식분할·병합, 자본감소, 해산, 이사·감사 해임(일반 → 특별 전환 케이스) |

정족수 판정은 **두 요건을 모두 충족**해야 가결입니다. 한쪽만 만족하면 부결.

## 정기주주총회 전형 안건 7종

정기주총은 **재무제표 승인**과 **배당 결의**가 핵심입니다. 비상장 중소기업에서 실제로 자주 다루는 3건(★)에 집중해도 무방합니다.

1. **★ 재무제표 승인의 건** — 상법 §449·§447 (대차대조표·손익계산서·이익잉여금처분계산서 등)
2. **★ 이익잉여금 처분(배당 결의)의 건** — §462 (배당 가능 이익 한도 내)
3. **★ 임원 보수한도 승인의 건** — §388 (정관에 금액 미정 시 주총 결의 필요)
4. 이사 선임의 건 — §382 (보통결의)
5. 감사 또는 감사위원 선임의 건 — §409 (감사는 의결권 3% 초과분 행사 제한)
6. 정관 일부 변경의 건 — §433·§434 (특별결의)
7. 주식매수선택권 부여의 건 — §340의2·§340의3 (정관 근거 + 주총 보통결의)

> 💡 **실무 권고**: 재무제표 승인과 배당결의는 **별도 안건으로 분리**하는 것이 분쟁 예방에 유리합니다. "재무제표를 승인하면 배당도 포함된다"는 해석은 실무상 다툼이 있습니다.

## 실무 안건 예시 문구 3종

### 예시 1 — 재무제표 승인 (보통결의)

> **제1호 의안: 제○○기(○○○○년 ○월 ○일 ~ ○○○○년 ○월 ○일) 재무제표 승인의 건**
>
> 의장은 제○○기 재무제표(대차대조표, 손익계산서, 이익잉여금처분계산서)를 별첨과 같이 제출하고 그 내용을 설명한 후 본 의안의 승인을 구한 바, 출석 주주 전원의 찬성으로 **원안대로 가결**되다.

### 예시 2 — 임원 보수한도 승인 (보통결의)

> **제2호 의안: 이사 및 감사의 보수한도 승인의 건**
>
> 의장은 제○○기 이사의 보수한도액을 금 ○○○,○○○,○○○원, 감사의 보수한도액을 금 ○○,○○○,○○○원으로 정할 것을 부의한 바, 출석 주주 전원의 찬성으로 **원안대로 가결**되다.

### 예시 3 — 이익배당 결의 (보통결의)

> **제3호 의안: 제○○기 이익잉여금 처분(현금배당) 결의의 건**
>
> 의장은 제○○기 이익잉여금 중 1주당 금 ○○○원, 총액 금 ○○○,○○○,○○○원을 현금배당하고, 배당기준일을 ○○○○년 ○월 ○일로 하며 배당금 지급 개시일을 ○○○○년 ○월 ○일로 할 것을 부의한 바, 출석 주주 전원의 찬성으로 **원안대로 가결**되다.

## CLI 사용 예시 (4모드)

계산기는 `python3 references/calculator.py <mode>` 형식이며, 출력은 JSON(`ensure_ascii=False, indent=2`)입니다.

### 1. `quorum` — 의결 정족수 판정

```bash
python3 references/calculator.py quorum \
  --type special \
  --total-shares 1000 \
  --present-shares 500 \
  --affirmative-shares 335
```

### 2. `notice-deadline` — 소집통지 마감일

```bash
python3 references/calculator.py notice-deadline \
  --meeting-date 2026-05-15 \
  --capital-under-1bn
```

### 3. `checklist` — 의사록 필수 기재 체크

```bash
python3 references/calculator.py checklist \
  --has-company-name --has-meeting-type --has-date-place \
  --has-attendance --has-chair --has-opening \
  --has-agenda-results --has-closing --has-signatures
```

### 4. `agenda-template` — 안건 문구 템플릿

```bash
python3 references/calculator.py agenda-template \
  --agenda-type financial-statement \
  --company-name "주식회사 엘비즈파트너스" \
  --fiscal-year "제10기"
```

지원 안건 11종 + `custom`:
`financial-statement` · `dividend` · `director-compensation` · `director-appointment` · `articles-amendment` · `stock-option` · `ceo-appointment` · `headquarters-relocation` · `capital-reduction` · `director-removal` · `merger` · `custom`

### 5. `full-minutes` — 회사 유형별 전체 의사록 텍스트

```bash
python3 references/calculator.py full-minutes \
  --company-type general \
  --meeting-type regular \
  --fiscal-year 10 \
  --meeting-date 2026-03-31 \
  --company-name "주식회사 엘비즈파트너스" \
  --address "서울시 강남구 테헤란로 123" \
  --agenda-types financial-statement,dividend,director-compensation
```

출력 JSON의 `template_text` 필드에 복붙 가능한 완전한 의사록 텍스트가 담깁니다.

## 회사 유형별 의사록 템플릿 (5종)

상수님 실무 템플릿을 기반으로 회사 형태별 5종 양식을 지원합니다. `full-minutes` 모드의 `--company-type` 옵션으로 선택합니다.

| 유형                 | 옵션값               | 핵심 특징                                                                                                       |
| -------------------- | -------------------- | --------------------------------------------------------------------------------------------------------------- |
| 비상장 일반          | `general`            | 상법 §363 2주 전 소집통지 → 의장·출석·안건·폐회 표준 흐름                                                       |
| 소규모(자본금 10억↓) | `small`              | "**자본금 총액 10억 원 미만 회사로서 상법 제363조에 따라 주주총회일 10일 전에 각 주주에게 소집통지를 완료**"    |
| 1인 주주 개최형      | `single-shareholder` | "**주주 전원의 동의에 따라 소집절차 없이 주주총회를 개최**" + "주주 ○○○가 의결권 있는 주식 ○○주 전부 보유·출석" |
| 1인 서면결의형       | `written-resolution` | 제목 "**주주총회 결의서**" + "**상법 제363조 제5항에 따라 아래 사항에 대하여 서면으로 동의**" — 개회·폐회 생략  |
| ⭐ 엘비즈 표준       | `lbiz-standard`      | 상수님 실무에서 실제 사용하시는 정기주총 의사록 양식 — 자간 띄어쓰기 제목, 기본 3안건(재무제표·보수한도·배당)   |

## ⭐ 엘비즈 표준 양식 (lbiz-standard)

상수님이 실무에서 실제 사용하시는 정기주총 의사록 양식입니다. 아드폰테스 2024년 정기주주총회 의사록 원본을 그대로 재현한 유형으로, `--company-type lbiz-standard` 옵션으로 호출합니다.

### 원본 양식 특징

- **제목**: "제 N 기 정 기 주 주 총 회 의 사 록" — 자간 띄어쓰기 적용
- **의장 등단**: "의장은 정각 ○시에 등단하여 개회를 선언하다." 고정 문구
- **이사 보수한도**: 대표이사 한도 + 이사(대표이사 외) 한도를 2단으로 분리 기재
- **정기배당**: 배당총액 + 배당기준일 + 지급시기까지 구체적으로 명시
- **별첨**: 별첨1(재무상태표·손익계산서·이익잉여금처분계산서) + 별첨2(정기배당 결의서) 2종

### 필수 파라미터

| 파라미터                  | 설명                    | 예시              |
| ------------------------- | ----------------------- | ----------------- |
| `--fiscal-year`           | 기수(숫자)              | `10`              |
| `--company-name`          | 회사명                  | `"○○"`            |
| `--meeting-date`          | 총회일(YYYY-MM-DD)      | `2026-03-15`      |

## 임시주주총회 템플릿

정기주총이 사업연도 결산·배당 중심이라면, **임시주주총회**는 특정 안건을 신속히 처리하는 총회입니다. 의사록상 **소집 경위**와 **안건의 특정성**이 중요합니다.

| 구분        | 정기주주총회                         | 임시주주총회                                                    |
| ----------- | ------------------------------------ | --------------------------------------------------------------- |
| 개최 주기   | 매 사업연도 종료 후 3개월 이내(정관) | 필요 시 수시                                                    |
| 핵심 안건   | 재무제표 승인, 배당, 임원 보수한도   | 이사 선임/해임, 대표이사 선임, 정관변경, 본점이전, 자본 감소 등 |
| 의사록 표기 | "제○○기 정기주주총회 의사록"         | "임시주주총회 의사록" (기수 표기 선택적)                        |

### 자주 쓰는 임시주총 안건 7종

1. 이사 선임(`director-appointment`)
2. 이사 해임(`director-removal`) — 특별결의
3. 대표이사 선임(`ceo-appointment`)
4. 정관 변경(`articles-amendment`) — 특별결의
5. 본점 이전(`headquarters-relocation`)
6. 자본금 감소(`capital-reduction`) — 특별결의
7. 주식매수선택권(`stock-option`) / 합병·분할·영업양도(`merger`) — 특별결의

### 호출 예시 (임시주총)

```bash
python3 references/calculator.py full-minutes \
  --company-type general --meeting-type extraordinary --fiscal-year 10 \
  --meeting-date 2026-08-15 \
  --agenda-types ceo-appointment,headquarters-relocation
```

## 되묻기 규칙

상수님이 "주주총회 의사록 만들어줘" 또는 "lbiz-standard 양식으로 작성해줘"라고 요청하면 아래 5단계 순서로 정보를 수집한 뒤 생성합니다.

### lbiz-standard 양식 요청 시 5단계 대화

1. **회사명 · 기수** — 회사명과 몇 기 정기주총인지 확인 (예: "제10기")
2. **일시(요일 포함) · 장소** — 총회 개최 날짜(요일 포함), 개회 시각, 폐회 시각, 장소
3. **출석현황** — 발행주식 총수, 주주 총수, 출석 주주 수, 출석 주식 수
4. **의장·대표이사 이름** — 동일인인지 확인 (다른 경우 각각 확인)
5. **기본 3안건 제시 후 가감 확인**:
   - 제1호 재무제표 승인 (기본 포함) — 결산연도 확인
   - 제2호 이사 보수한도 (대표이사 한도 + 이사 한도 각각 입력 필요)
   - 제3호 정기배당 (총액 + 배당기준일 + 지급시기 입력 필요)
   - "빼거나 추가할 안건 있나요? (예: 정관변경, 이사 선임)"

> 💡 **소집 기간 단축 동의서** 필요 여부도 함께 확인할 것 (자본금 10억 미만 회사인 경우 자동 제안).

### 재무제표 업로드 로드맵 안내 (향후)

현재는 수동 입력 방식입니다. 추후 엑셀(xlsx) 파싱으로 재무상태표·손익계산서·이익잉여금처분계산서 수치를 자동으로 채워 넣는 기능을 추가할 예정입니다.

## 안건별 표준 문구 예시 (상수님 제공)

### ① 이사 선임 (보통결의)

> **제○호 의안: 이사 선임의 건**
>
> 의장은 이사 ○○○를 선임하고자 함을 설명한바, **상법 및 정관이 정한 결의요건을 충족하였으므로 출석주주의 의결권의 과반수와 발행주식총수의 4분의 1 이상 찬성으로 원안대로 승인 가결하다.**

### ② 정관 변경 (특별결의)

> **제○호 의안: 정관 일부 변경의 건**
>
> 의장은 정관 일부 변경안을 별첨과 같이 제출하고 주요 변경사항을 설명한바, **상법 및 정관이 정한 결의요건을 충족하였으므로 출석주주의 의결권의 3분의 2 이상과 발행주식총수의 3분의 1 이상 찬성으로 원안대로 승인 가결하다.**

### ③ 본점 이전 (보통결의 — 정관 범위 내)

> **제○호 의안: 본점 이전의 건**
>
> 의장은 회사 본점을 ○○시 ○○구 ○○로 ○○로 이전하고자 함을 설명한바, **상법 및 정관이 정한 결의요건을 충족하였으므로 출석주주의 의결권의 과반수와 발행주식총수의 4분의 1 이상 찬성으로 원안대로 승인 가결하다.**

> ⚠️ 본점 이전이 **정관상 본점 소재지 변경**을 수반하면 **정관 변경(특별결의)** 안건을 함께 상정해야 합니다.

## 작성 포인트 (상수님 실무 팁)

- **배당결의**: "1주당 배당액 + 배당기준일 + 지급 개시일"까지 구체적으로 기재 권장. 금액만 적으면 후속 실무에서 다툼 소지.
- **재무제표 승인**: "**재무상태표·손익계산서·이익잉여금처분계산서(안)**"을 명시 — 구 표현("대차대조표")도 허용되나 현행 K-IFRS 표현 통일이 안전.
- **임원 보수한도**: 이사 보수한도와 감사 보수한도를 **분리 기재** 권장 (감사 독립성 확보).
- **등기 연결 안건**(이사 선임·해임, 대표이사 선임, 정관변경, 본점이전, 합병 등): 법무사·공증 단계에서 지적받지 않도록 **등기용 문구로 엄격히** 다듬어야 함. 공증이 필요한 안건은 의사록 원본 + 인감증명 + 주주명부 등 부속서류 사전 점검.
- **의사록 마무리**: **의장 + 출석 이사** 기명날인 또는 서명이 §373③ 필수 요건. 의장만 서명하면 효력 다툼 소지 — 출석 이사 전원 서명란 누락 금지.

## 실무 주의사항

- **재무제표 승인과 배당결의는 별도 안건으로 분리**하는 것이 분쟁 예방에 유리.
- **결의 결과 표현을 일관되게**: "원안대로 가결" · "만장일치 가결" · "출석 주주 ○○%의 찬성으로 가결" 중 하나로 통일.
- **이사·감사 선임 등 이해관계 안건**에서 기권·반대표가 있는 경우 **반드시 별도 기재**. "전원 찬성"으로 뭉뚱그리면 후일 분쟁 근거가 됨.
- **전자투표(§368의4)·서면투표(§368의3)** 사용 시 해당 사실과 집계 내역을 의사록에 반영.
- **감사 선임**에서 대주주는 **3%룰**(§409②) 적용됨을 유의 — 의결권 집계를 별도로 수행.
- **소규모회사 특례**(자본금 10억↓ + 이사 1명): 이사회가 없으므로 이사회 결의사항을 주총에서 처리하는 경우가 많음. 등기 관련 실익 확인 필수.

## 알려진 한계 (Known Limitations)

1. **상장회사 특례 미반영** — 상장회사 감사위원 분리선출(§542의11·§542의12), 소집통지 공고 방식, 전자공시 의무 등은 본 스킬 범위 밖.
2. **주주명부 폐쇄·기준일 실무** — §354 주주명부 폐쇄 공고, 의결권 기준일 설정은 별도 검토 필요.
3. **등기·공증 요건** — 이사 변경·정관변경 등 등기사항은 등기소 제출 서류·공증(공증인법) 별도 확인. 본 스킬은 의사록 내용만 다룸.
4. **서면결의(§363④)** 주주 전원의 서면동의로 결의한 경우의 의사록 형식은 별도 (본 스킬은 실제 총회 개최 전제).
5. **외국인 주주·의결권 대리행사** 관련 특수 절차 미반영.
6. **벤처기업법·조세특례제한법** 상 주식매수선택권(§15·§15의2) 특례는 본 스킬의 일반 §340의2 기준으로 다룸.
