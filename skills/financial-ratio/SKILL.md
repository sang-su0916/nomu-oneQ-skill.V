# 재무비율 분석기 (financial-ratio)

⚠️ **반드시 계산기 CLI를 실행하세요. 직접 계산 금지** — hallucination 방지.

## 언제 사용

- 기업 재무제표(재무상태표·손익계산서)에서 주요 비율을 한꺼번에 산출할 때
- 유동성·안정성·수익성·활동성 중 특정 카테고리만 빠르게 점검할 때
- 부채비율·이자보상배율·ROE 등으로 한계기업·재무위험 여부를 판단할 때
- 컨설팅·심사·대출 검토 전 재무건전성 1차 스크리닝

## 법령 근거

| 근거                  | 내용                                                        | 출처                                                            |
| --------------------- | ----------------------------------------------------------- | --------------------------------------------------------------- |
| 일반 재무회계 원리    | 재무비율 정의 (유동·당좌·부채·ROE·ROA·자산회전율 등)        | K-IFRS / 일반기업회계기준                                       |
| 금융감독원 가이드     | 재무비율 표준치 및 한계기업 판정 기준 (이자보상배율 1배 등) | [금융감독원 fss.or.kr](https://www.fss.or.kr) (2026-04-14 조회) |
| 한국은행 기업경영분석 | 업종별 재무비율 평균 공표                                   | [한국은행 ecos.bok.or.kr](https://ecos.bok.or.kr)               |

## 핵심 공식

```
─ 유동성 ──────────────────────────────────────────────────
유동비율       = 유동자산 / 유동부채 × 100 (%)
당좌비율       = (유동자산 − 재고자산) / 유동부채 × 100 (%)
현금비율       = 현금및현금성자산 / 유동부채 × 100 (%)

─ 안정성 (레버리지) ──────────────────────────────────────
부채비율       = 총부채 / 자기자본 × 100 (%)
부채대자산     = 총부채 / 총자산 × 100 (%)
자기자본비율   = 자기자본 / 총자산 × 100 (%)
이자보상배율   = 영업이익(EBIT) / 이자비용 (배)

─ 수익성 ──────────────────────────────────────────────────
매출순이익률   = 당기순이익 / 매출 × 100 (%)
영업이익률     = 영업이익 / 매출 × 100 (%)
ROA           = 당기순이익 / 총자산 × 100 (%)
ROE           = 당기순이익 / 자기자본 × 100 (%)

─ 활동성 ──────────────────────────────────────────────────
총자산회전율   = 매출 / 총자산 (회)
재고자산회전율 = 매출원가 / 재고자산 (회)
매출채권회전율 = 매출 / 매출채권 (회)
매출채권회수일 = 365 / 매출채권회전율 (일)
재고자산회전일 = 365 / 재고자산회전율 (일)
```

## 표준치 해석 기준 (제조업 기준)

| 비율         | 양호          | 보통       | 주의·위험           |
| ------------ | ------------- | ---------- | ------------------- |
| 유동비율     | 200% 이상     | 150%~200%  | 150% 미만           |
| 당좌비율     | 100% 이상     | 70%~100%   | 70% 미만            |
| 부채비율     | 100% 이하     | 100%~200%  | 200% 이상 위험      |
| 자기자본비율 | 50% 이상      | 30%~50%    | 30% 미만            |
| 이자보상배율 | 3배 이상      | 1~3배      | 1배 미만 (한계기업) |
| ROE          | 10% 이상 우수 | 5~10% 보통 | 5% 미만 저조        |
| ROA          | 5% 이상 양호  | 2~5% 보통  | 2% 미만 저조        |

⚠️ **업종별로 표준치가 크게 다름.** 제조업·서비스업·IT·금융업 간 직접 비교 불가. 동종업계 평균과 비교해야 의미 있음.

## CLI 사용법

### `liquidity` — 유동성 비율

```bash
python3 skills/financial-ratio/references/calculator.py liquidity \
  --current-assets 150000000 \
  --current-liabilities 100000000 \
  --inventory 50000000 \
  --cash-and-equiv 30000000
```

### `leverage` — 안정성·부채 비율

```bash
python3 skills/financial-ratio/references/calculator.py leverage \
  --total-debt 300000000 \
  --total-equity 200000000 \
  --total-assets 500000000 \
  --ebit 30000000 \
  --interest-expense 10000000
```

### `profitability` — 수익성 비율

```bash
python3 skills/financial-ratio/references/calculator.py profitability \
  --net-income 15000000 \
  --revenue 500000000 \
  --total-assets 500000000 \
  --total-equity 200000000 \
  --operating-income 25000000
```

### `activity` — 활동성 비율

```bash
python3 skills/financial-ratio/references/calculator.py activity \
  --revenue 500000000 \
  --total-assets 500000000 \
  --inventory 50000000 \
  --accounts-receivable 80000000 \
  --cogs 350000000
```

### `overall` — 종합 진단 (4개 카테고리 + 경보 플래그)

```bash
python3 skills/financial-ratio/references/calculator.py overall \
  --current-assets 150000000 --current-liabilities 100000000 \
  --total-debt 300000000 --total-equity 200000000 --total-assets 500000000 \
  --ebit 30000000 --interest-expense 10000000 \
  --net-income 15000000 --revenue 500000000 --operating-income 25000000 \
  --inventory 50000000 --accounts-receivable 80000000 --cogs 350000000
```

출력: liquidity / leverage / profitability / activity 4개 섹션 + `summary`(종합 문장) + `flags`(위험경보 배열).

## 입력 파싱 가이드

| 사용자 입력 예                          | 액션             | 비고                          |

## 되묻기 규칙 (정보 부족 시)

1. **재무제표 시점 확인**: "기말(12월 31일) 기준인가요, 평균(기초+기말)/2 기준인가요? 일반적으로 회전율·ROA/ROE는 평균 자산을 사용합니다."
2. **연결 vs. 별도**: "연결재무제표 기준인가요, 별도재무제표 기준인가요? 자회사 지분이 많을 경우 결과가 크게 달라집니다."
3. **업종 확인**: "제조업 기준 표준치로 해석해드립니다. 서비스·IT·금융업이라면 업종별 표준치가 달라 비교 해석에 주의가 필요합니다."
4. **매출원가 부재**: "재고자산회전율 계산에는 매출원가(COGS)가 필요합니다. 없으면 매출로 대체 가능하지만 과대평가됩니다."
5. **영업이익 vs. EBIT**: "이자보상배율은 일반적으로 영업이익(EBIT) 기준입니다. 세전이익을 EBIT로 혼동하지 마세요."

## 도메인 특수 규칙

- **업종 차이 극심**: 금융업은 부채비율이 본질적으로 높음(자기자본 대비 예수금). 단순 표준치 대입 금지.
- **회전율 기간 일치**: 연간 매출 ÷ 기말 자산 대신 연간 매출 ÷ 평균 자산이 이론적으로 정확 (본 계산기는 입력값을 그대로 사용).
- **이자보상배율 1배 미만 연속 3년**: 금융감독원 한계기업 정의 — 본 계산기는 단년 기준만 판정.
- **당좌자산 ≠ 유동자산 − 재고자산** 엄밀 정의: 당좌자산 = 현금+유가증권+매출채권 등. 본 계산기는 편의상 (유동자산 − 재고)로 근사.
- **K-IFRS vs. K-GAAP**: 리스부채·금융부채 분류 차이로 부채비율이 달라질 수 있음.

## 응답 포맷

계산 결과에 다음 항목을 포함하세요.

- 입력 재무제표 요약
- 카테고리별 비율 (%, 소수점 2자리)
- 각 비율에 대한 표준치 비교 해석
- 경보 플래그 (`flags` 배열의 warning 항목)
- 종합 요약 (overall 모드)
- 면책 문구

**면책**: 본 계산은 일반 재무회계 원리에 따른 참고치입니다. 업종·기업규모·회계기준(K-IFRS vs. K-GAAP)·연결범위에 따라 해석이 달라질 수 있습니다. 실제 투자·대출·M&A 판단은 회계사·재무전문가 검토가 필요합니다.

## 알려진 한계

1. **업종별 표준치 미반영** — 제조업 기준 단일 해석. 금융·서비스·IT·건설업은 별도 표준치 참조 필요 (한국은행 ECOS 기업경영분석).
2. **단년·단일 시점 분석** — 추세(3~5년 YoY) 및 동종업계 평균 대비 비교는 수동 분석 필요. 한계기업 판정(이자보상 1배 미만 3년 연속)도 단년 기준 경보만 제공.
3. **평균 자산·재고 미지원** — 회전율·ROA/ROE는 엄밀히 (기초+기말)/2 기준이나, 본 계산기는 입력값(통상 기말)을 그대로 사용.
4. **비재무 정보 미반영** — 경영진 교체, 업황, 수주잔고, 영업현금흐름 등 질적 요소 제외. 재무비율만으로는 기업 부실을 모두 포착 못함.
5. **K-IFRS 조정 없음** — 리스부채, 금융자산 분류, 유효이자율법 등 회계기준 차이는 입력값에 그대로 반영됨. 입력 전 재무제표 기준을 통일할 것.
