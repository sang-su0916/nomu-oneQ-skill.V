# 재무진단 종합 스킬 (financial-diagnosis)

⚠️ **반드시 계산기 CLI를 실행하세요. 직접 계산 금지** — hallucination 방지.

## 언제 사용

- 재무제표 전체를 입력받아 "한 장 종합 리포트"가 필요할 때
- 0-100 스코어·S/A/B/C/D 등급으로 의사결정 자료를 만들 때
- 한계기업·재무위험 기업 1차 스크리닝
- 컨설팅 착수 보고, 대출 심사 전 내부 검토, M&A 실사 전 예비 진단
- 동종업계 다수 기업을 동일 기준으로 비교해야 할 때

## 포함 지표

본 스킬은 `financial-ratio`, `break-even`, `depreciation` 3개 스킬의 핵심 공식을 통합하여 단일 JSON 리포트를 생성합니다.

| 카테고리           | 핵심 비율                                    | 가중치 |
| ------------------ | -------------------------------------------- | ------ |
| 유동성 (liquidity) | 유동비율·당좌비율·현금비율                   | 25%    |
| 안정성 (leverage)  | 부채비율·자기자본비율·이자보상배율           | 30%    |
| 수익성 (profit.)   | 매출순이익률·영업이익률·ROA·ROE              | 25%    |
| 활동성 (activity)  | 총자산·재고·매출채권 회전율 및 회수일        | (보조) |
| 이자보상           | EBIT ÷ 이자비용 (한계기업 지표)              | 20%    |
| (선택) BEP         | 손익분기점 수량·매출 (fixed-cost 등 입력 시) | -      |

## 스코어링 로직

각 지표를 100점 만점으로 환산 후 가중평균:

| 지표         | 100점     | 70-75점  | 40-60점  | 20-25점   |
| ------------ | --------- | -------- | -------- | --------- |
| 유동비율     | 200% 이상 | 150-199% | 100-149% | 100% 미만 |
| 부채비율     | 100% 이하 | 100-200% | 200-300% | 300% 초과 |
| ROE          | 10% 이상  | 5-10%    | 0-5%     | 적자      |
| 이자보상배율 | 3배 이상  | 1-3배    | -        | 1배 미만  |

**등급 환산**:

| 점수    | 등급     | 의미                               |
| ------- | -------- | ---------------------------------- |
| 90+     | S (탁월) | 재무 초우량                        |
| 75-89   | A (우수) | 재무 건전, 투자·대출 대상으로 양호 |
| 60-74   | B (양호) | 보통 수준, 일부 개선 항목 존재     |
| 45-59   | C (주의) | 재무위험 주의, 개선 계획 필수      |
| 45 미만 | D (위험) | 한계기업 근접·재무구조 개선 시급   |

## 위험 플래그 규칙

- **한계기업 근접**: 이자보상배율 < 1배
- **재무구조 심각**: 부채비율 ≥ 300%
- **단기 지급불능 위험**: 유동비율 < 100%
- **적자기업**: ROE < 0

## 권장사항 생성 규칙

| 조건                     | 권장사항                               |
| ------------------------ | -------------------------------------- |
| 부채비율 200% 초과       | 자본확충 또는 부채 감축 필요           |
| 유동비율 100% 미만       | 단기 유동성 확보 시급                  |
| 이자보상배율 1배 미만    | 영업활동 이자상환 불가 — 한계기업 위험 |
| ROE 5% 미만              | 수익성 개선 필요 (원가절감·마진 확대)  |
| 매출채권회수일 60일 초과 | 매출채권 회수기일 단축 필요            |
| 재고자산회전율 4회 미만  | 재고 과다 점검 필요                    |

## CLI 사용법

### `diagnose` — 종합 재무진단

```bash
python3 skills/financial-diagnosis/references/calculator.py diagnose \
  --current-assets 150000000 --current-liabilities 100000000 \
  --inventory 50000000 --accounts-receivable 80000000 \
  --total-debt 300000000 --total-equity 200000000 --total-assets 500000000 \
  --revenue 500000000 --net-income 15000000 --operating-income 25000000 --cogs 350000000 \
  --ebit 30000000 --interest-expense 10000000
```

BEP 통합 분석이 필요하면 선택 인자 추가:

```bash
  --fixed-cost 100000000 --unit-price 10000 --unit-variable-cost 6000
```

출력 JSON 주요 필드:

- `diagnosis_date`: 진단일 (YYYY-MM-DD)
- `liquidity` / `leverage` / `profitability` / `activity`: 카테고리별 상세 결과
- `break_even`: BEP 입력 시만 포함
- `overall_score`: 0-100 종합 점수
- `grade`: S/A/B/C/D + 의미
- `score_breakdown`: 카테고리별 환산 점수
- `strengths` / `weaknesses`: 강점·약점 배열
- `risk_flags`: 위험 경보 배열
- `recommendations`: 권장사항 배열
- `summary`: 1-2문장 종합 요약
- `disclaimer`: 면책 문구

## 입력 파싱 가이드

| 사용자 입력 예                        | 액션                           |
| ------------------------------------- | ------------------------------ |
| "이 회사 재무제표 보고 종합 진단해줘" | `diagnose`                     |
| "등급 매겨줘, S/A/B/C/D로"            | `diagnose`                     |
| "한계기업인지 판단해줘"               | `diagnose` + 이자보상배율 확인 |
| "BEP까지 같이 넣어서 종합 리포트"     | `diagnose` + BEP 선택 인자     |

## 되묻기 규칙

1. **기말 vs. 평균**: "재무상태표 수치는 기말 기준인가요, (기초+기말)/2 평균 기준인가요? 회전율·ROA/ROE 정확도가 달라집니다."
2. **연결 vs. 별도**: "연결 기준인가요 별도 기준인가요? 자회사 지분이 많으면 결과가 크게 달라집니다."
3. **EBIT 확인**: "이자보상배율은 영업이익(EBIT) 기준입니다. 세전이익과 혼동하지 마세요."
4. **업종 확인**: "표준치는 제조업 기준입니다. 금융·건설·서비스업은 별도 해석이 필요합니다."
5. **BEP 입력**: "손익분기점까지 포함하시려면 고정비·단위판매가·단위변동비가 필요합니다."

## 도메인 특수 규칙

- **본 스킬은 meta-wrapper**: `financial-ratio` 4개 카테고리 + `break-even` 옵션을 한 번에 호출.
- **subprocess 대신 직접 구현**: 안정성을 위해 공식을 복사(중복)했습니다. financial-ratio 변경 시 동기화 필요.
- **단년 기준**: 한계기업 정식 정의(이자보상 1배 미만 3년 연속)는 반영 못함 — 단년 경보만 제공.
- **업종 표준치 단일**: 제조업 기준 가중치 고정. 금융업 부채비율은 본질적으로 높아 D등급 왜곡 가능.

## 응답 포맷

계산 결과를 다음 순서로 제시하세요.

1. 종합 점수 + 등급 (굵게)
2. 카테고리별 핵심 비율 표
3. 강점 / 약점 (bullet)
4. 위험 플래그 (있을 때만)
5. 권장사항 (bullet, 우선순위 순)
6. 면책 문구

**면책**: 본 진단은 재무비율 단년 기준 참고치이며, 업종·회계기준(K-IFRS vs. K-GAAP)·연결범위에 따라 해석이 달라집니다. 실제 투자·대출·M&A 판단은 회계사·재무전문가 검토가 필요합니다.

## 알려진 한계

1. **공식 복제**: `financial-ratio/references/calculator.py`와 동일한 공식을 복사했습니다. 원본 수정 시 본 스킬도 동기화해야 합니다.
2. **단년·단일 시점**: 3-5년 추세, 연속 적자 판정 미지원.
3. **제조업 단일 표준**: 업종별 가중치·기준치 자동 전환 없음.
4. **비재무 정보 배제**: 경영진, 업황, 수주잔고, 영업현금흐름 미반영.
5. **스코어 가중치 고정**: 25/30/25/20 고정 — 용도별(대출·투자·내부관리) 가중치 조정 기능 없음.
