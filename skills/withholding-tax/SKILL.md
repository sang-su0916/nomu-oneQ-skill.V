# 원천징수 계산기 (withholding-tax)

⚠️ **반드시 계산기 CLI를 실행하세요. 직접 계산 금지** — hallucination 방지.

## 언제 사용

- 프리랜서·외주 용역 대금 지급 전 사업소득 3.3% 원천징수액 산정
- 강연료·원고료·인세·번역료 지급 시 기타소득 원천징수 (60% 필요경비 의제 후 20%)
- 경품·시상금·위자료 등 기타소득 유형별 의제율 확인
- 이자·배당소득 지급 시 15.4% 원천징수 계산 (2천만원 이하 분리과세 확인 포함)
- 일용근로자 일당 지급 시 일급 15만원 공제 후 원천세 계산
- 상용 근로자 월급 지급 시 간이세액표 조회 URL 안내

## 법령 근거

| 조문                  | 내용                                                                 | 출처                                                |
| --------------------- | -------------------------------------------------------------------- | --------------------------------------------------- |
| 소득세법 §129①1호·2호 | 이자·배당소득 원천징수세율 14% (+지방세 1.4% = 15.4%)                | [국가법령정보센터 law.go.kr](https://www.law.go.kr) |
| 소득세법 §129①3호     | 사업소득(인적용역·의료·개인서비스) 원천징수 3% (+지방세 0.3%=3.3%)   | law.go.kr                                           |
| 소득세법 §129①6호     | 기타소득 원천징수 20% (+지방세 소득세의 10% = 실효 22% of 과세표준)  | law.go.kr                                           |
| 소득세법 §21          | 기타소득 범위 (강연료·원고료·경품·위자료·사례금 등)                  | law.go.kr                                           |
| 소득세법 시행령 §87   | 기타소득 필요경비 의제율 (강연료·원고료 60%, 공익법인 시상금 80% 등) | law.go.kr                                           |
| 소득세법 §47          | 일용근로자 근로소득공제 (일 15만원)                                  | law.go.kr                                           |
| 소득세법 §134         | 근로소득 원천징수 (상용: 간이세액표, 일용: 6% × (1-0.55))            | law.go.kr                                           |
| 지방세법 §92          | 지방소득세 = 소득세액의 10%                                          | law.go.kr                                           |

## 핵심 공식

```
─ 사업소득 (§129①3호) ─────────────────────────────────────────
소득세   = 지급금액 × 3%
지방세   = 지급금액 × 0.3%
원천징수 = 지급금액 × 3.3%

─ 기타소득 (§21·§129①6호·시행령 §87) ──────────────────────────
필요경비 = 지급금액 × 의제율 (유형별)
과세표준 = 지급금액 - 필요경비
소득세   = 과세표준 × 20%
지방세   = 소득세 × 10%
원천징수 = 소득세 + 지방세

※ "기타소득 20% 고정" 아님 — 유형별 필요경비율이 달라 실효율 다름.

─ 이자·배당 (§129①1호·2호) ───────────────────────────────────
소득세   = 지급액 × 14%
지방세   = 지급액 × 1.4%
원천징수 = 지급액 × 15.4%

─ 일용근로자 (§47·§134) ──────────────────────────────────────
과세표준 = max(0, 일급 - 150,000)
소득세   = 과세표준 × 6% × (1 - 0.55 세액공제)
         = 과세표준 × 2.7%
지방세   = 소득세 × 10%

※ "일용 실효 2.7%"는 과세표준 대비 고정 — 지급총액 대비가 아님.

─ 상용 근로자 (§134 + 시행령 §194 별표) ───────────────────────
간이세액표(월급·부양가족·20세 이하 자녀수) 조회
→ 본 스킬은 홈택스 간이세액표 URL 안내만 제공
```

## 기타소득 유형별 필요경비 의제율 (시행령 §87)

| 유형 코드 | 적용 대상                                  | 의제율 | 실효율 (payment 대비) |
| --------- | ------------------------------------------ | ------ | --------------------- |
| `lecture` | 강연료·원고료·인세·번역료·방송출연료       | 60%    | 약 8.8%               |
| `prize`   | 공익법인 시상금 등                         | 80%    | 약 4.4%               |
| `general` | 위자료·사례금·복권 당첨금·기타 (의제 없음) | 0%     | 22% (과세표준=지급액) |
| `custom`  | 사용자 지정 의제율                         | 변동   | 계산 결과             |

※ 경륜·경마 환급금, 공익법인 상금 일부 등 세부 규정은 시행령 §87 각 호 확인 필요.

## CLI 사용법

### `business` — 사업소득 원천징수

```bash
python3 skills/withholding-tax/references/calculator.py business \
  --payment 1000000
```

출력: `income_tax` 30,000 / `local_tax` 3,000 / `total_withholding` 33,000 / `net_payment` 967,000.

### `other` — 기타소득 원천징수

```bash
# 강연료 100만원 (60% 필요경비 의제)
python3 skills/withholding-tax/references/calculator.py other \
  --payment 1000000 --type lecture

# 공익법인 시상금
python3 skills/withholding-tax/references/calculator.py other \
  --payment 5000000 --type prize

# 위자료·사례금 (필요경비 없음)
python3 skills/withholding-tax/references/calculator.py other \
  --payment 2000000 --type general

# 사용자 지정 의제율
python3 skills/withholding-tax/references/calculator.py other \
  --payment 1000000 --type custom --expense-rate 0.5
```

출력: `deemed_expense_rate`, `deemed_expense`, `taxable_income`, `income_tax`, `local_tax`, `total_withholding`, `effective_rate`.

### `interest-dividend` — 이자·배당 원천징수

```bash
python3 skills/withholding-tax/references/calculator.py interest-dividend \
  --amount 10000000
```

출력: `income_tax` 1,400,000 / `local_tax` 140,000 / `total_withholding` 1,540,000.

### `daily-worker` — 일용근로자 원천징수

```bash
# 일급 20만 × 1일
python3 skills/withholding-tax/references/calculator.py daily-worker \
  --daily-wage 200000

# 일급 25만 × 5일
python3 skills/withholding-tax/references/calculator.py daily-worker \
  --daily-wage 250000 --days 5
```

출력: `taxable_per_day`, `income_tax_per_day`, `local_tax_per_day`, `total_per_day`, `total_days_withholding`, `effective_rate_per_day`, `exempt_from_withholding`.

### `employment` — 상용 근로소득 안내

```bash
python3 skills/withholding-tax/references/calculator.py employment \
  --monthly-wage 3000000 --dependents 1
```

출력: 간이세액표 안내 + 홈택스 URL (계산 미수행).

## 되묻기 규칙 (정보 부족 시)

1. **소득 구분 확인**: "어떤 소득 유형인가요? ①사업소득(프리랜서 용역·의료) ②기타소득(강연·원고·경품·위자료) ③이자·배당 ④일용근로(일당제) ⑤상용근로(월급)."
2. **기타소득 유형 확인**: "기타소득 중 구체적인 유형을 알려주세요. 강연료·원고료·인세·방송출연료는 60% 필요경비, 공익법인 시상금은 80%, 위자료·사례금·복권은 필요경비 없음입니다."
3. **지급금액 vs 순지급액**: "지급 총액(세전) 기준인가요, 실지급액(세후) 기준인가요? 본 계산기는 지급 총액 입력 → 원천세 차감 후 실지급액 산출입니다."
4. **일용 vs 상용**: "일용근로자인가요(근로계약 3개월 미만·일당 지급), 상용근로자인가요(정기 급여)? 일용은 일급 15만 공제 후 단일 계산, 상용은 간이세액표 조회입니다."
5. **이자·배당 2천만 초과 여부**: "연간 금융소득 2천만원을 초과하나요? 초과 시 종합과세 대상이라 원천징수 15.4%는 예납 성격이 됩니다."

## 도메인 특수 규칙

- **②기타소득 20% 고정 금지** — 유형별 필요경비 의제율이 다르므로 반드시 type 확인.
- **⑤일용근로 실효율은 계산 결과** — "2.7%"는 과세표준 대비 고정이고, 지급총액 대비 실효율은 일급마다 다름. (예: 일급 18만 → 실효율 약 0.495%, 일급 30만 → 실효율 약 1.485%)
- **사업소득 면세사업자 구분**: 부가세 면세사업자(의료·교육·예술 등)에게 지급하는 대가가 사업소득으로 분류될 때 3.3% 원천징수. 과세사업자는 세금계산서 발행 → 원천징수 대상 아님.
- **기타소득 연 300만원 이하 선택적 분리과세**: 기타소득금액(필요경비 차감 후) 연 300만원 이하 시 원천징수로 납세의무 종결 선택 가능 (소득세법 §14③).
- **연말정산 차액 정산**: 상용 근로자 간이세액표 원천징수는 예납 — 연말정산에서 실제 결정세액과 차액 정산.
- **지방소득세 별도 납부처**: 원천징수의무자는 소득세(세무서)와 지방소득세(지자체)를 각각 다른 곳에 납부.

## 알려진 한계

1. **상용 근로소득 간이세액표 미탑재** — 월급·부양가족·자녀수 기반 간이세액표가 방대해 본 스킬은 홈택스 URL 안내로 처리. 정확한 금액은 홈택스 계산기 조회 필수.
2. **4대보험 공제 미반영** — 근로소득 원천징수 계산 시 실무에서는 4대보험료 공제 후 과세표준 산정. 본 스킬은 원천세만 산정하며, 4대보험은 `four-insurances` 스킬 사용.
3. **기타소득 세부 시행령 분기 미완비** — 경륜·경마 환급금, 광업권 양도, 저작권 양도 등 시행령 §87 각 호 세부 의제율은 `custom` 으로 수동 입력 필요.
4. **연 300만원 이하 분리과세 선택 미반영** — 선택적 분리과세 여부는 본인 판단, 본 계산기는 무조건 원천징수 산정.
5. **지급조서 제출 의무 미안내** — 원천징수의무자는 지급조서 제출 의무 (다음연도 2/3/5월 등 구분). 본 스킬 범위 외.
6. **외국인 거주자 특례 미반영** — 외국인 근로자 단일세율 19% 특례(조세특례제한법 §18의2) 등은 별도.

## 응답 포맷

계산 결과에 다음 항목을 포함하세요.

- 원천징수 유형 (사업소득 / 기타소득 / 이자·배당 / 일용 / 상용)
- 지급총액 (입력값 재확인)
- 필요경비 (해당 시) + 의제율
- 과세표준 (해당 시)
- 소득세 + 지방세 + 총 원천징수액
- 순지급액 (= 지급총액 - 총 원천징수)
- 실효율 (지급총액 대비)
- 산식 + 법령 근거
- 면책 문구

**면책**: 2026년 기준 원천징수 안내. 기타소득 필요경비 의제율은 시행령 §87 지정 유형에 한함. 상용 근로소득은 간이세액표(홈택스) 사용 필수. 일용근로자 실효율은 일급에 따라 달라지는 계산 결과이지 고정 세율 아님. 실무 원천징수 이행은 세무 전문가 확인 권장.

## 관련 스킬

- `income-tax` — 종합소득세 (원천징수는 예납 성격, 5월 종합소득세 신고에서 정산)
- `corporate-tax-interim-payment` — 법인세 중간예납 (법인은 원천징수 수취 시 기납부세액 반영)
- `value-added-tax` — 부가가치세 (과세사업자는 세금계산서 발행, 원천징수 대상 아님 ↔ 면세사업자 대가는 사업소득 원천)
- `severance-pay` — 퇴직소득은 별도 원천징수 체계 (퇴직소득세 연분연승법)
- `four-insurances` — 상용 근로자 월급 원천세 계산 시 4대보험료 공제 베이스
- `wage-base` — 일용·상용 근로자 임금 산정 베이스
- `omsc` — 본 스킬은 OMSC scaffold 기반 세무 도메인 스킬 (노무 연관: 일용·상용 근로소득)
