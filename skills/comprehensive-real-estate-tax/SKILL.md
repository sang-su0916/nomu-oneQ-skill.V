# 종합부동산세 계산기 (comprehensive-real-estate-tax)

⚠️ **반드시 계산기 CLI를 실행하세요. 직접 계산 금지** — hallucination 방지.

⚠️ **주택분 기준 스킬**: 토지(종합합산·별도합산)분 종부세는 세율·공제 체계가 달라 본 스킬 범위 밖.

⚠️ **공정시장가액비율 단정 금지**: 시행령 §2의4①로 정해진 **현재값 60%**를 적용하되, 법상 60~100% 범위(§8②)에서 시행령 개정에 따라 변동 가능. 응답 시 "시행령 현재값 60%" 구조로 표현.

## 법령 근거

| 조문                   | 내용                                                      | 출처                        |
| ---------------------- | --------------------------------------------------------- | --------------------------- |
| 종부세법 §8            | 주택분 과세표준 = (공시가 합산 − 공제) × 공정시장가액비율 | law.go.kr (2026-04-14 조회) |
| 종부세법 §8①           | 공제액: 1세대1주택자 12억 / 그 외 9억 / 법인 0            | law.go.kr                   |
| 종부세법 §8②           | 공정시장가액비율 60~100% 범위에서 대통령령 위임           | law.go.kr                   |
| 종부세법 §9①           | 주택분 세율 (2주택 이하 / 3주택 이상 7단계)               | law.go.kr                   |
| 종부세법 §9②3          | 법인 단일세율 2.7% (2주택 이하) / 5.0% (3주택 이상)       | law.go.kr                   |
| 종부세법 §9③           | 재산세 중복분 공제 (계산 편의상 별도 안내)                | law.go.kr                   |
| 종부세법 §9⑥⑦⑧⑨        | 1세대1주택자 고령자·장기보유 세액공제 (합산 80% 한도)     | law.go.kr                   |
| 종부세법 시행령 §2의4① | 공정시장가액비율 현재값 60% (2022년 이후 현행)            | law.go.kr                   |

## 핵심 공식

```
과세표준 = max(공시가격 합산 − 공제액, 0) × 공정시장가액비율(60%)
기본세액 = 과세표준 × 주택수별 세율표 적용 (2주택 이하 / 3주택 이상)
법인세액 = 과세표준 × 단일세율 (2주택 이하 2.7% / 3주택 이상 5.0%)
1세대1주택 납부세액 = 기본세액 × (1 − min(고령자공제 + 장기보유공제, 80%))
```

## 공제액 (§8①)

| 구분            | 공제액 |
| --------------- | -----: |
| 1세대1주택자    |   12억 |
| 그 외 개인      |    9억 |
| 법인 (중과대상) |      0 |

## 공정시장가액비율 (§8② / 시행령 §2의4①)

- **법상 범위**: 주택분 60~100% (대통령령 위임)
- **시행령 현재값**: **60%** (2022년 이후 현행)
- ⚠️ 시행령 개정으로 변동 가능 — 세무사·국세청 확인 필수

## 세율표 — 주택분 2주택 이하 (§9①)

| 과세표준 구간         | 세율 |
| --------------------- | :--: |
| 3억 이하              | 0.5% |
| 3억 초과 ~ 6억 이하   | 0.7% |
| 6억 초과 ~ 12억 이하  | 1.0% |
| 12억 초과 ~ 25억 이하 | 1.3% |
| 25억 초과 ~ 50억 이하 | 1.5% |
| 50억 초과 ~ 94억 이하 | 2.0% |
| 94억 초과             | 2.7% |

## 세율표 — 주택분 3주택 이상 (§9①)

12억 이하는 2주택과 동일, **12억 초과부터 중과**.

| 과세표준 구간         |   세율   |
| --------------------- | :------: |
| 3억 이하              |   0.5%   |
| 3억 초과 ~ 6억 이하   |   0.7%   |
| 6억 초과 ~ 12억 이하  |   1.0%   |
| 12억 초과 ~ 25억 이하 | **2.0%** |
| 25억 초과 ~ 50억 이하 | **3.0%** |
| 50억 초과 ~ 94억 이하 | **4.0%** |
| 94억 초과             | **5.0%** |

## 법인 단일세율 (§9②3)

| 구분       | 세율 |
| ---------- | :--: |
| 2주택 이하 | 2.7% |
| 3주택 이상 | 5.0% |

## 실전 대화 예시

### 예시 1 — 2주택자 개인 (공시 합산 30억)

**사용자**: "아파트 두 채 갖고 있는데 공시가 합쳐서 30억이에요. 종부세 얼마나 나올까요?"

**응답 흐름**:

1. `household --published-price 3000000000` 실행
2. 과세표준 = (30억 − 9억) × 60% = **12.6억**
3. 세액 계산: 12억까지 누적 960만 + (0.6억 × 1.3%) = **1,038만원**
4. 재산세 중복 공제(§9③) 및 세부담상한(§10)은 별도 안내

### 예시 2 — 3주택자 개인 (동일 공시 합산 30억)

**사용자**: "아파트 세 채 공시가 합쳐서 30억이면 얼마예요? 2주택일 때보다 많이 나와요?"

**응답 흐름**:

1. `multi-home --published-price 3000000000` 실행
2. 과세표준 = 12.6억 (동일)
3. 12억 초과분 0.6억 × **2.0%** 중과 세율 적용 = 960만 + 120만 = **1,080만원**
4. 2주택(1,038만) 대비 3주택(1,080만) = 약 42만원 증가

### 예시 3 — 1세대1주택 고령·장기보유 (공시 20억)

**사용자**: "1주택 공시가 20억이에요. 저 올해 70세고, 15년째 같은 집 살고 있어요."

**응답 흐름**:

1. `single-home-senior --published-price 2000000000 --age 70 --holding-years 15` 실행
2. 과세표준 = (20억 − 12억) × 60% = **4.8억**
3. 기본세액 = 150만 + (1.8억 × 0.7%) = **276만원**
4. 고령자 40% + 장기보유 50% = 90% → **80% 한도 적용**
5. 납부세액 = 276만 × 20% = **55.2만원**
6. 한도 80% 적용 표시 (`credit_cap_applied: true`)

## 되묻기 규칙 (정보 부족 시)

1. **주택 수 불명**: "1세대 기준 몇 채 보유하고 계시나요? 1주택(12억 공제·특례), 2주택 이하(9억 공제·일반세율), 3주택 이상(9억 공제·12억 초과 중과)로 구분됩니다."
2. **개인·법인 구분**: "개인이신가요, 법인이신가요? 법인은 공제 0원·단일세율(2.7%/5.0%)이 적용됩니다."
3. **1세대1주택 특례 대상**: "1세대 1주택자(본인 명의 단독, 배우자·세대원과 합산)이신가요? 공제 12억 + 고령자·장기보유 세액공제(중복 80% 한도)는 1세대1주택자에게만 적용됩니다."
4. **연령·보유기간**: "고령자 공제(60·65·70세)와 장기보유 공제(5·10·15년)는 과세기준일(6월 1일) 시점 기준입니다."
5. **공시가격**: "부동산공시가격알리미(realtyprice.kr)에서 올해 공시가격 확인 부탁드립니다. 시가와 다르고, 재산세·종부세 과세 기준은 공시가격입니다."

## 도메인 특수 규칙

- **과세기준일**: 매년 **6월 1일** 현재 보유 주택 기준
- **공정시장가액비율 단정 금지**: 시행령 §2의4①로 **현재 60%**이나, 법상 60~100% 범위 내에서 시행령 개정으로 변동 가능 — 응답 시 "현재값 60%" 표현 유지
- **세부담상한 (§10)**: 직전연도 총세액의 150% 한도 — 본 계산기 미반영
- **재산세 중복 공제 (§9③)**: 동일 과세대상 주택에 이미 납부한 재산세 해당분 차감 — 본 계산기 미반영
- **합산배제**: 임대주택·사원용 주택·미분양주택 등은 주택분 합산에서 제외 (§8②단서, 시행령 §3) — 본 계산기 미반영
- **세액공제 중복 80% 한도**: 고령자 공제 + 장기보유 공제 합산이 80%를 초과해도 80% cap 적용 (§9⑨)
- **지방세는 별도**: 종부세는 국세 — 지방소득세 10% 가산 없음 (재산세는 별도 지방세)

## 응답 포맷

계산 결과에 다음 항목을 포함하세요.

- 공시가격 합산 (입력값 재확인)
- 공제액 (12억/9억/0 — 구분 명시)
- 공정시장가액비율 (**"시행령 현재값 60% — 법상 60~100% 범위"** 명시)
- 과세표준
- 적용 세율표 (2주택 이하 / 3주택 이상 / 법인)
- 기본세액 + 세액공제 내역 (1세대1주택 특례 시) + 납부세액
- 법령 근거 (§8/§9 및 시행령 §2의4)
- 면책 문구 + 재산세 중복 공제·세부담상한 별도 안내

**면책**: 2026년 기준 종합부동산세법. 공정시장가액비율은 시행령 §2의4①로 정해진 현재값 60%이며, 법상 60~100% 범위(§8②)에서 시행령 개정에 따라 변동 가능. 공시가격 실제 확인은 부동산공시가격알리미(realtyprice.kr) 별도 조회 필수. 재산세 중복 공제(§9③), 세부담상한(§10), 부속토지·종업원 주거·임대주택 합산제외 등은 미반영. 실제 신고 시 국세청·세무 전문가 확인 필수.

## 알려진 한계 (Known Limitations)

1. **공정시장가액비율 변동 가능** — 시행령 §2의4로 현재 60%, 법상 60~100% 범위. 시행령 개정 시 본 계산기 상수 업데이트 필요.
2. **공시가격 직접 조회 필요** — 부동산공시가격알리미(realtyprice.kr) 별도 확인. 본 계산기는 입력값 기반.
3. **재산세 중복 공제 미반영** (§9③) — 동일 과세대상 주택에 이미 납부한 재산세 해당분 차감은 실무상 별도 계산.
4. **세부담상한 미반영** (§10) — 직전연도 총세액의 150% 한도는 본 계산기 미반영.
5. **합산제외 주택 미반영** — 부속토지·종업원 주거·임대주택·미분양주택·기숙사·어린이집 등 합산배제 대상 주택은 본 계산기 범위 밖.
6. **토지분 미지원** — 종합합산토지·별도합산토지 종부세는 별개 세율 체계로 본 스킬 범위 밖.
7. **1세대 판정 복잡성** — 세대원 중 다른 명의 주택이 있으면 1세대1주택 특례 대상에서 제외. 실제 판정은 등기부·주민등록 교차 확인 필요.
8. **법인 예외** — 공익법인·공공주택사업자 등 법인 중 일부는 단일세율이 아닌 일반세율 적용(§9②단서) — 본 계산기 미반영.
