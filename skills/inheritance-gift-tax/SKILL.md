# 상속세·증여세 계산기 (inheritance-gift-tax)

⚠️ **반드시 계산기 CLI를 실행하세요. 직접 계산 금지** — 구간·한도·공제 hallucination 방지.

⚠️ **거주자 기준 스킬**: 비거주자·법인 증여, 가업상속공제·영농상속공제·동거주택 상속공제 특례는 본 스킬 범위 밖.

## 법령 근거

| 조문            | 내용                                                                               | 출처                        |
| --------------- | ---------------------------------------------------------------------------------- | --------------------------- |
| 상증세법 §18    | 기초공제 2억원 (상속세)                                                            | law.go.kr (2026-04-14 조회) |
| 상증세법 §19    | 배우자 상속공제 — min(법정상속분 한도액, 30억), 실제상속 5억 미만 시 5억           | law.go.kr                   |
| 상증세법 §21    | 일괄공제 — max(기초+기타인적공제, 5억). 배우자 단독상속은 불가                     | law.go.kr                   |
| 상증세법 §26    | 상속세·증여세 5단계 누진세율 (10~50%)                                              | law.go.kr                   |
| 상증세법 §53    | 증여재산공제 — 10년 합산 (배우자 6억 / 직계 5천만 / 미성년 2천만 / 기타친족 1천만) | law.go.kr                   |
| 상증세법 §53의2 | 혼인·출산공제 — 직계존속, 혼인·출산일 전후 2년, **③항 합산 1억 한도**              | law.go.kr                   |

## 세율표 (상증세법 §26, 5단계 — 상속·증여 동일)

| 과세표준 구간                             | 세율 |  누진공제액 |
| ----------------------------------------- | :--: | ----------: |
| 100,000,000원 이하                        | 10%  |           0 |
| 100,000,000원 초과 ~ 500,000,000 이하     | 20%  |  10,000,000 |
| 500,000,000원 초과 ~ 1,000,000,000 이하   | 30%  |  60,000,000 |
| 1,000,000,000원 초과 ~ 3,000,000,000 이하 | 40%  | 160,000,000 |
| 3,000,000,000원 초과                      | 50%  | 460,000,000 |

## 증여재산공제 (상증세법 §53, 10년 합산)

| 관계              | 공제 한도 | 비고                |
| ----------------- | --------: | ------------------- |
| 배우자            |       6억 | —                   |
| 직계존속 → 성년   |     5천만 | 부모·조부모 등      |
| 직계존속 → 미성년 |     2천만 | `--is-minor` 플래그 |
| 직계비속          |     5천만 | 자녀·손자녀 등      |
| 기타친족          |     1천만 | 4촌 혈족, 3촌 인척  |

## 혼인·출산공제 (상증세법 §53의2)

- **직계존속으로부터만** 적용 (§53과 별개로 추가)
- **혼인공제**: 혼인신고일 전후 2년 이내 증여 → 1억 한도
- **출산공제**: 자녀 출생·입양일부터 2년 이내 증여 → 1억 한도
- **⚠️ ③항 핵심: 혼인공제 + 출산공제 합산 1억 한도** (둘 다 받아도 최대 1억, 중복 불가)

## CLI 사용법

### `inheritance` — 상속세 계산

```bash
# 상속재산 20억, 배우자공제 5억, 일괄공제 5억 → 과세표준 10억 → 산출 2.4억
python3 skills/inheritance-gift-tax/references/calculator.py inheritance \
  --estate 2000000000 \
  --spouse-deduction 500000000 \
  --lump-sum-deduction 500000000
```

| 플래그                 | 기본값      | 설명                             |
| ---------------------- | ----------- | -------------------------------- |
| `--estate`             | 필수        | 상속재산가액 (원)                |
| `--spouse-deduction`   | 0           | 배우자 상속공제 (§19, 30억 한도) |
| `--lump-sum-deduction` | 500,000,000 | 일괄공제 (§21, 5억)              |
| `--basic-deduction`    | 200,000,000 | 기초공제 (§18, 2억)              |
| `--other-deduction`    | 0           | 기타인적공제 (일괄공제와 비교)   |

### `gift` — 증여세 계산

```bash
# 성년 자녀에게 직계존속이 1억 증여 → §53 공제 5천만 → 과세 5천만 → 산출 500만
python3 skills/inheritance-gift-tax/references/calculator.py gift \
  --gift-amount 100000000 --relation lineal-descendant
```

| 플래그                   | 기본값            | 설명                                                           |
| ------------------------ | ----------------- | -------------------------------------------------------------- |
| `--gift-amount`          | 필수              | 증여재산가액 (원)                                              |
| `--relation`             | lineal-descendant | spouse / lineal-ascendant / lineal-descendant / other-relative |
| `--is-minor`             | flag              | 미성년 수증자 (직계존속 → 2천만)                               |
| `--prior-deduction-used` | 0                 | 직전 10년 내 공제 사용액                                       |

### `marriage-birth` — 혼인·출산공제 시뮬레이션

```bash
# 혼인 전 직계존속 증여 1.5억 → §53 5천만 + §53의2 혼인 1억 = 1.5억 공제 → 과세 0
python3 skills/inheritance-gift-tax/references/calculator.py marriage-birth \
  --gift-amount 150000000 --marriage-deduction 100000000

# 합산 한도 초과 (혼인 1억 + 출산 1억) → §53의2③ 1억 한도 적용, warnings 출력
python3 skills/inheritance-gift-tax/references/calculator.py marriage-birth \
  --gift-amount 250000000 --marriage-deduction 100000000 --birth-deduction 100000000
```

### `compare` — 상속 vs 증여 비교

```bash
python3 skills/inheritance-gift-tax/references/calculator.py compare \
  --estate 1000000000 --spouse-deduction 500000000 --relation lineal-descendant
```

## 입력 파싱 가이드

| 사용자 입력 예                      | 액션                                | 비고                                |
| ----------------------------------- | ----------------------------------- | ----------------------------------- |
| "상속재산 20억 배우자·자녀 2명"     | `inheritance` 즉시                  | 배우자 법정상속분 계산 되묻기       |
| "부모님이 1억 증여하면 세금"        | `gift --relation lineal-descendant` | §53 5천만 공제 후 500만             |
| "결혼 전에 1억 받아도 되나요"       | `marriage-birth`                    | §53 5천만 + §53의2 1억 = 1.5억 공제 |
| "혼인공제 1억 + 출산공제 1억 둘 다" | `marriage-birth` 즉시               | **③항 1억 한도 안내 필수**          |
| "상속이 나아요 증여가 나아요"       | `compare` 즉시                      | 단순 비교 한계 안내                 |

## 되묻기 규칙 (정보 부족 시)

1. **거주자·비거주자 구분**: "피상속인·수증자가 거주자인가요? 비거주자는 본 스킬 범위 밖입니다 (별도 계산)."
2. **배우자 공제**: "배우자 법정상속분 한도 적용 시 다른 상속인 수·지분 확인 필요. 실제상속액이 5억 미만이면 5억 공제."
3. **10년 합산**: "증여는 직전 10년 내 동일인으로부터 받은 금액을 합산. 과거 증여액을 알려주시면 `--prior-deduction-used`로 반영."
4. **혼인·출산 일자**: "§53의2는 혼인·출산일 전후 2년 이내만. 시기 벗어나면 §53만 적용."
5. **특례 상속공제**: "가업상속공제·영농상속공제·동거주택 상속공제는 본 스킬 미반영. 해당 시 별도 검토."

## 도메인 특수 규칙

- **상속 vs 증여 비교 한계**: 단순 세액 비교로 결정 금지. 자산 종류·합산과세·사후관리·가업승계·감정평가 종합 고려
- **10년 합산과세 미반영**: 본 계산기는 단발성 증여 기준. 직전 10년 내 증여액은 `--prior-deduction-used`로 수동 반영
- **§53의2 ③항 우선**: 혼인+출산 합산 1억 한도는 자동 적용되며, 초과 요청 시 `warnings` 필드 출력
- **배우자 단독상속 시 일괄공제 불가**: §21② 단서 — 배우자 단독상속은 일괄공제 적용 제외 (현재 계산기는 수동 조정 필요)
- **상속세 연대납부**: 상속인 간 연대납부 의무 존재 (납세의무자 구분은 별도 협의)
- **증여세 신고기한**: 증여받은 날이 속하는 달의 말일부터 3개월 이내 신고·납부

## 실전 대화 예시

**예시 1 — 상속세**

> "아버지가 20억 남기셨어요. 어머니와 자녀 2명인데 세금 얼마 나와요?"

```bash
# 법정상속분: 배우자 1.5/3.5, 자녀 각 1/3.5 → 배우자 법정상속액 ≈ 8.57억
# 일괄공제 5억 + 배우자공제 5억 (또는 법정상속분 한도) 가정
python3 skills/inheritance-gift-tax/references/calculator.py inheritance \
  --estate 2000000000 --spouse-deduction 500000000 --lump-sum-deduction 500000000
# 과세표준 10억 → 산출세액 2.4억 (10억×30% - 6천만)
```

**예시 2 — 결혼 앞두고 증여**

> "결혼 앞두고 부모님이 1.5억 주신다는데 세금?"

```bash
python3 skills/inheritance-gift-tax/references/calculator.py marriage-birth \
  --gift-amount 150000000 --marriage-deduction 100000000
# §53 직계존속 5천만 + §53의2 혼인 1억 = 1.5억 공제 → 과세표준 0 → 세금 0원
# (단, 혼인신고일 전후 2년 이내 요건 충족 필요)
```

**예시 3 — 혼인+출산공제 중복 시도**

> "혼인공제 1억에 출산공제 1억까지 총 2억 공제 가능한가요?"

```bash
python3 skills/inheritance-gift-tax/references/calculator.py marriage-birth \
  --gift-amount 250000000 --marriage-deduction 100000000 --birth-deduction 100000000
# ⚠️ §53의2③ 합산 1억 한도 → 실제 공제 1억 (2억 불가)
# 총공제 1.5억 (§53 5천만 + §53의2 1억) → 과세표준 1억 → 산출 1천만
```

## 알려진 한계

1. **상증세법 개편 불확실성** — 2026년 현재 정부·국회 논의 진행 중 (유산취득세 전환·일괄공제 상향·자녀공제 재편 등). 본 계산기는 2026-04-14 법제처 조회 기준 현행 법령 기준.
2. **비거주자 제외** — 거주자 기준만 반영. 비거주자 상속세(국내 소재 재산만 과세) 별도.
3. **10년 합산과세 단순화** — `--prior-deduction-used`로 공제한도 차감만 반영. 직전 증여재산가액 합산과세(§47) 세액 재계산은 미반영.
4. **특례 공제 미반영** — 가업상속공제(§18의2), 영농상속공제(§18의3), 동거주택 상속공제(§23의2), 특별연부공제 등 제외.
5. **법정상속분 자동 산정 미지원** — 배우자공제 한도는 사용자가 수동 입력 (법정상속분 × 상속재산 계산은 별도).
6. **배우자 단독상속 일괄공제 예외** — §21② 단서 조항 자동 판정 미지원.
7. **신고·납부 절차 미반영** — 연부연납·물납, 신고세액공제(3%), 무신고·과소신고 가산세 등 제외.
8. **지방세 없음** — 상속세·증여세는 국세 (종합소득세와 달리 지방소득세 부과 없음).
