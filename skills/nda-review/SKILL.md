# NDA 검토 (nda-review)

> ⚠️ **본 체크리스트는 1차 스크리닝 도구입니다.** 실제 체결·분쟁 대응은 **변호사·변리사 자문이 필수**입니다. 영업비밀 침해는 민사·형사 양방향 리스크가 큽니다.

NDA(Non-Disclosure Agreement, 비밀유지계약서) 전문을 제공하면, 부정경쟁방지법 §2 영업비밀 3요건을 기준으로 **필수 10조항 충족 여부**와 **독소 조항 5종**을 검토합니다.

## 영업비밀 3요건 (NDA 검토의 출발점)

부정경쟁방지법 §2 2호는 다음 **3요건을 모두 충족한 정보만** "영업비밀"로 보호합니다. **하나라도 결여되면 형사처벌(§18)·손해배상(§14의2) 모두 불가**합니다.

```
① 비공지성 (Secrecy)
   — 일반에 알려져 있지 않은 상태
   — 인터넷·업계 표준 문서로 이미 공개된 정보는 ❌

② 경제적 가치성 (Economic Value)
   — 정보 자체가 경쟁력의 원천
   — 실제 사용/잠재적 사용가치가 있어야 함

③ 비밀관리성 (Reasonable Efforts to Maintain Secrecy)
   — 상당한 노력으로 비밀로 유지
   — 증거 예: "CONFIDENTIAL" 표기, 접근권한 통제, 보안교육 기록,
             출입 제한, 패스워드, 서약서, 전산 로그
   — 관리 흔적이 없으면 법원에서 "비밀"로 인정 안 됨
```

**검토 포인트**: NDA 조항에 **① 비밀정보 표기 방식**, **② 접근권한자 지정**, **③ 물리적·전자적 관리 의무**가 명시되어야 3요건 중 "비밀관리성"을 뒷받침할 수 있습니다.

## NDA 필수 명시사항 10가지 (체크리스트)

각 항목을 ✅(충분) · ⚠️(미흡) · ❌(누락/위반)로 판정합니다.

| #   | 항목                     | 검토 포인트                                                                                               | 체크 |
| --- | ------------------------ | --------------------------------------------------------------------------------------------------------- | ---- |
| 1   | **당사자 특정**          | 법인명·사업자등록번호·대표자 성명·주소. 외국법인은 본점 소재지 + 한국 내 송달주소                         | ☐    |
| 2   | **비밀정보의 정의·범위** | 추상적 "영업상 정보" 지양. **구체적 유형 열거** (고객명단·원가표·기술문서·소스코드·설계도·마케팅계획 등)  | ☐    |
| 3   | **비밀정보의 표기·전달** | "CONFIDENTIAL" 마킹·봉함·암호화·열람권한자 지정. 구두 제공분은 사후 서면화 기한 명시(통상 30일)           | ☐    |
| 4   | **사용목적 제한**        | 본 검토·협상·공동 프로젝트 목적으로만 사용. 그 외 사용·복제·재공개·역공학·제3자 전달 금지                 | ☐    |
| 5   | **반환·폐기 의무**       | 계약종료 또는 제공자 요청 시 **전량 반환 또는 폐기** + **폐기확인서** 서면 제출                           | ☐    |
| 6   | **비밀유지 기간**        | 계약 종료 후 **X년** (통상 3~5년). 영업비밀이 진정한 비밀인 경우 무기한도 가능하나 **합리적 한도** 필요   | ☐    |
| 7   | **예외 사유**            | 이미 공지된 정보 / 제3자로부터 적법 취득한 정보 / 자체 독립 개발 정보 / 법령·법원·감독기관 공개 의무 정보 | ☐    |
| 8   | **위반 시 손해배상**     | 손해배상 근거 명시. **구체적 손해액 입증** vs **위약벌** 선택 (위약벌은 민법 §398 감액 대상 유의)         | ☐    |
| 9   | **관할·준거법**          | 대한민국 법 준거. 관할: **서울중앙지방법원** 또는 합의 관할 + 중재(대한상사중재원) 선택                   | ☐    |
| 10  | **서명·날인**            | 대표자 서명 + 법인 인감(또는 사용인감 + 사용인감계). 개인은 자필 서명 + 신분증 사본                       | ☐    |

**평가 기준**:

- ✅ **충분**: 항목이 명확하고 구체적으로 기재
- ⚠️ **미흡**: 항목이 있으나 모호하거나 부분적 (예: "관련 정보 일체")
- ❌ **누락/위반**: 항목이 없거나 강행법규 위반

## 불공정 조항 스캔 (5종 독소 조항)

다음 조항이 발견되면 **❌** 또는 **⚠️**로 표시하고 수정 권고합니다.

### 1. 과도한 위약벌 ❌

**금지 패턴**:

- "위반 시 금 5억원을 즉시 지급한다"
- "손해액 입증 없이 10억 지급 의무"

**법적 근거**: 민법 §398④ — 손해배상액의 예정이 과다한 경우 **법원이 감액** 가능. 명목상 천문학적 위약벌은 §103(반사회질서) 또는 약관규제법 위반으로 무효·축소됩니다.

**권고**: 구체적 손해액 입증 조항 + 합리적 위약금 상한 (통상 계약금액의 1~3배).

### 2. 편무적 비밀유지 ⚠️

**금지 패턴**:

- 제공자는 정보만 받고 의무 無, 수령자만 일방적 비밀유지
- "을은 갑의 모든 정보를 영구 비밀로 한다. 갑은 본 계약에 구속되지 아니한다"

**권고**: **상호 NDA(Mutual)**로 전환하거나, 일방 NDA라도 **제공자의 정보 정확성 보증·권리 침해 없음 보증**을 추가.

### 3. 영구 기간 (Perpetual) ⚠️/❌

**금지 패턴**:

- "본 NDA는 기간 제한 없이 영구 유효하다"
- 영업비밀이 아닌 일반 정보에까지 무기한 제한

**권고**:

- 일반 정보: **3~5년** 이내
- 진정한 영업비밀(기술·원가 등): 무기한 가능하나 "해당 정보가 **공지될 때까지**" 조건 추가
- 영구 기간 + 광범위 정보 범위 → 민법 §103 무효 사유

### 4. 광범위 경쟁금지 끼워넣기 ❌

**금지 패턴**:

- NDA 내에 **경업금지(Non-compete)** 조항 삽입
- "본 NDA 체결 후 동종업계 2년간 근무·창업 금지" (별도 대가 없이)

**법적 근거**: 대법원은 경업금지 조항에 **합리적 범위**(기간·지역·직종)와 **별도 대가** 요구. NDA에 끼워넣는 포괄 경쟁금지는 무효 판례 다수.

**권고**: 경업금지는 **별도 계약**으로 분리 + 합리적 대가 지급.

### 5. 포괄 면책 ❌

**금지 패턴**:

- "제공자는 본 정보에 대한 일체의 책임을 지지 아니한다"
- "정보의 정확성·완전성·비침해성에 대해 어떤 보증도 하지 않는다" (고의·중과실까지 면책)

**법적 근거**: 민법 §103·약관규제법 §7 — **고의·중과실 면책 조항은 무효**.

**권고**: 경과실 면책은 가능하나 **고의·중과실은 제외** 명시.

## 계약 유형별 추가 체크

| 유형                           | 추가 검토 포인트                                                                                                |
| ------------------------------ | --------------------------------------------------------------------------------------------------------------- |
| **상호 NDA** (Mutual)          | 양 당사자 의무 대칭성. 사용 목적 양방향 명시. 정보 분류(어느 쪽 정보인지) 구분                                  |
| **일방 NDA** (One-way)         | 수령 당사자 의무 집중. 제공자 권리 보호 (표기·반환·폐기 의무 강화)                                              |
| **M&A 실사용** (Due Diligence) | **Data Room 접근 통제** (로그·워터마크·다운로드 금지). 딜 종료 후 **반환·삭제 확인서**. 클린팀(Clean Team) 지정 |
| **용역 NDA**                   | 위탁업무 특정. **하도급·재위탁 제한** 또는 사전 동의 절차. 용역 종료 후 성과물 귀속 별도 명시                   |
| **연구협력 NDA**               | **IP 귀속** 별도 명시 필수 (공동 발명·단독 발명 구분). 출판·발표 제한 조항. 공동 연구 성과 배분                 |

## 검토 워크플로 (Claude 수행 절차)

```
STEP 1: 사용자가 NDA 텍스트 제공
        (전문 복사 / 사진·PDF 업로드)

STEP 2: 당사자·계약 유형·사용 목적 파악
        (되묻기 규칙 적용)

STEP 3: 필수 10개 항목 ✅/⚠️/❌ 판정

STEP 4: 독소 조항 5종 스캔 (위약벌·편무·영구·경쟁금지·포괄면책)

STEP 5: 계약 유형별 추가 체크
        (상호/일방/M&A/용역/연구협력)

STEP 6: 결과 리포트 작성
        - 10조항 체크 표 + 독소 조항 표
        - 최종 평가 (완전/보완/위반)
        - 수정 권고 + 다음 단계
```

## 되묻기 규칙 (정보 부족 시)

| 부족 정보          | 재질문                                                                                 |
| ------------------ | -------------------------------------------------------------------------------------- |
| 당사자 역할 불명   | "본 NDA에서 **정보 제공자**인가요, **수령자**인가요? (양쪽 다라면 상호 NDA)"           |
| 계약 유형 불명     | "**상호 NDA / 일방 NDA / M&A 실사용 / 용역 NDA / 연구협력 NDA** 중 어디에 해당하나요?" |
| 비밀정보 종류 불명 | "보호 대상 정보가 **기술·영업·재무·인사·고객정보** 중 어떤 것인가요?"                  |
| 기간 불명          | "비밀유지 기간이 **몇 년**인가요? (계약서 본문 또는 부속서에서 확인)"                  |
| 본계약 관계 불명   | "**독립 NDA**인가요, **본 계약(용역·M&A 등)의 부속 서류**인가요?"                      |

다음 구조로 응답합니다.

```markdown

| --- | ------------- | ---- | -------------------------------- |
| 1   | 당사자 특정   | ✅   | 법인명·대표자·주소 명확          |
| 2   | 비밀정보 범위 | ⚠️   | "영업상 정보 일체" — 구체화 필요 |
| ... |

**미흡·누락 항목**: [번호 나열] → 보완 요청

### 독소 조항 5종 스캔

| #   | 조항          | 판정    | 비고                                 |
| --- | ------------- | ------- | ------------------------------------ |
| 1   | 과도한 위약벌 | ❌ 발견 | "위반 시 10억" — 민법 §398 감액 대상 |
| ... |

**위반 조항**: [나열] → 삭제/수정 권고

### 계약 유형별 추가 체크

(해당하는 경우 M&A·용역·연구협력 등 추가 점검)

### 최종 평가

- **완전**: 10조항 모두 충분 + 독소 조항 없음 → "서명 가능"
- **보완**: 미흡 항목 있음 → "상대방에 수정 요청 후 서명"
- **위반**: 독소 조항 발견 → "해당 조항 삭제 또는 서명 거부 권장"

### 다음 단계 권고

1. [구체적 수정 조항 안내]
2. [분쟁 대응 방법 — 민사 §14의2 vs 형사 §18]
3. [변호사 자문이 필요한 쟁점]
```

## 알려진 한계

1. **판례 최신 동향 미반영** — 2024~2026년 대법원 영업비밀 침해 판례, 부정경쟁방지법 개정 동향 미반영. 최신 판결 별도 확인 권장.
2. **산업별 특수 관행 미반영** — IT(소스코드·오픈소스 라이선스 충돌), 바이오(FDA·식약처 공시 의무), 금융(자본시장법 공시) 등 산업별 특수 규정 미포함.
3. **영문 NDA 미지원** — 국문 NDA 전용. 영문 NDA(영미법·CISG 등) 준거는 별도 변호사 자문 필수.
4. **구체적 손해액 산정 불가** — §14의2 추정 손해액은 개별 사안별(침해자 이익·통상 사용료) 판단이 필요하므로 본 체크리스트로는 금액 산출 불가.
5. **국제계약 미지원** — 준거법이 외국법(뉴욕주법·영국법 등)이거나 ICC·SIAC 중재 선택 시 별도 검토 필요.
