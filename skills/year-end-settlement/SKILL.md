# 연말정산 간이 추정 (year-end-settlement)

⚠️ **반드시 계산기 CLI를 실행하세요. 직접 계산 금지** — 구간·한도·공제율 hallucination 방지.

## 법령 근거 (2026 귀속)

| 조문             | 내용                                                         | 출처                        |
| ---------------- | ------------------------------------------------------------ | --------------------------- |
| 소득세법 §47     | 근로소득공제 (구간별 누적, 한도 2천만)                       | law.go.kr (2026-04-14 조회) |
| 소득세법 §50·§51 | 인적공제 (본인·부양가족 150만 / 경로 +100만 / 장애인 +200만) | law.go.kr                   |
| 소득세법 §55     | 기본세율 (8단계 누진, 2023년 귀속부터 현행)                  | nts.go.kr                   |
| 소득세법 §59     | 근로소득세액공제 (산출 130만↓ 55% / 초과 30%)                | law.go.kr                   |
| 소득세법 §59의2  | 자녀세액공제 (25/30/40만, 2026 인상)                         | law.go.kr                   |

| 조특법 §95의3    | 결혼세액공제 (부부 합산 100만, 2024~2026 신설)               | law.go.kr                   |
| 조특법 §126의2   | 신용카드 등 소득공제 (25% 초과분, 15/30/40%)                 | law.go.kr                   |
| 지방세법 §92     | 지방소득세 (결정세액 × 10%)                                  | law.go.kr                   |

## 핵심 흐름

```
총급여
  → 근로소득공제 (§47)
  → 근로소득금액
  → (인적공제 + 연금보험료 + 주택청약 + 신용카드 소득공제)
  → 과세표준
  → 기본세율 (§55)
  → 산출세액
  → 세액공제 (근로·자녀·연금계좌·의료·교육·월세·결혼·표준)
  → 결정세액
  → 지방소득세 (결정세액 × 10%)
  → 환급 / 추가납부 (기납부세액 - 총부담)
```

## 2026 귀속 주요 변경사항 (요약)

1. **결혼세액공제 신설** — 부부 합산 100만원, 2024~2026 혼인신고자
2. **자녀세액공제 확대** — 첫째 15→25만, 둘째 20→30만, 셋째 이상 30→40만/인
3. **월세 확대** — 대상 총급여 7천만→8천만, 한도 750만→1,000만, 공제율 17%/15%
4. **주택청약 확대** — 한도 240만→300만, 공제율 40%, 배우자 포함
5. **6세 이하 자녀 의료비 한도 폐지**
6. **체육시설 문화비 공제 신설** — 헬스장·수영장 30% (2025.7~)
7. **고향사랑기부금 한도 확대** — 500만→2,000만원
8. **출산지원금 비과세 확대**
9. **초등 2학년 이하 예체능 학원비 교육비공제** 추가

## 되묻기 규칙

1. **총급여 vs 과세표준 혼동**: "총급여(비과세 제외 연봉)인가요, 과세표준(공제 후)인가요? 본 스킬은 총급여 입력."
2. **자녀 나이**: "자녀세액공제는 만 8~20세만 해당. 다른 연령 자녀는 인적공제(부양가족)에만 포함."
3. **월세 요건**: "무주택 세대주·총급여 8천만 이하·법정 임차주택 요건. 미해당 시 공제 불가."
4. **연금저축 vs IRP**: "납입액 합산 한도 900만, 연금저축 단독 600만. 초과분 공제 불가."

## 간이 산정 공식 (핵심)

1. **근로소득공제** = 구간 누적 `(70/40/15/5/2%)`, 한도 2,000만
2. **인적공제** = 1,500,000 × (1 + 부양가족) + 1,000,000 × 경로 + 2,000,000 × 장애인
3. **연금보험료공제** = `min(총급여 × 4.5%, 2,700,000)` (국민연금 본인부담 간이)
4. **주택청약 소득공제** = `min(납입, 3,000,000) × 40%` (총급여 7천만↓)
5. **신용카드 소득공제** = `max(0, (사용액 - 총급여 × 25%) × 15%)`, 한도 300만 (간이)
6. **과세표준** = `근로소득금액 - 인적공제 - 연금보험료 - 주택청약 - 신용카드`
7. **산출세액** = 기본세율 8단계 적용 (§55)
8. **세액공제**:
   - 근로 = `min(산출×55% or 130만×55%+초과×30%, 총급여별 한도)`
   - 자녀 = `25만(첫째) + 30만(둘째) + 40만×(셋째↑)`
   - 연금 = `min(납입, 900만) × (총급여 5,500만↓ 16.5% / ↑ 13.2%)`
   - 의료 = `max(0, 의료비 - 총급여×3%) × 15%`, 한도 70만
   - 교육 = `교육비 × 15%`
   - 월세 = `min(월세, 1,000만) × (5,500만↓ 17% / ↑ 15%)`, 총급여 8천만↓만
   - 결혼 = 본인 50만
   - 표준 = 130,000
9. **결정세액** = `max(0, 산출 - 세액공제 합)`
10. **지방소득세** = 결정 × 10%
11. **환급·추납** = 기납부세액 - (결정 + 지방)

## 도메인 특수 규칙

- **근로소득자 한정**: 사업·기타소득만 있으면 5월 종합소득세 신고 대상 (`income-tax` 스킬)
- **분리과세 제외**: 퇴직금·양도소득·복권 등 종합과세 대상 아님
- **간소화 서비스**: 1월 15일 홈택스 개통, 수정·누락 자료 직접 등록 가능
- **경정청구**: 신고 종료 후 5년 내 (과다납부 환급 청구)

## 알려진 한계

1. **결정세액 세부 공제 단순화** — 특별소득공제(건강·고용보험료, 주택자금), 기부금·정치자금, 보장성보험료, 자녀출산·입양세액공제, 월세 상세 요건(주택 크기·보증금) 미반영
2. **신용카드 공제 단순화** — 체크·현금 30%, 전통시장·대중교통·문화비 각 300만 추가 한도 미반영 (간이는 신용 15% 기준 한도 300만만 적용)
3. **연금저축·IRP 구분 미반영** — 합산 한도 900만으로 통합 적용, 퇴직연금 통산 미반영
4. **국민연금 본인부담 추정** — 실제 월 소득월액 기준 보수월액 상·하한(2026년 약 617만/37만) 세부 미반영
5. **중도입사·중도퇴사 정산 미지원** — 근속월수 보정 없음
6. **월세 세액공제 요건** — 무주택 세대주·임차주택 전용면적·보증금 요건 별도 확인 필요
7. **분리과세 특례 미반영** — 주택임대 분리과세(2천만 이하), 금융소득 분리과세 등

## 관련 스킬

- `income-tax` — 종합소득세 (기본세율 §55 공유, 근로 외 소득 병행 시)
- `withholding-tax` — 원천징수 (기납부세액 산정 근거)
- `four-insurances` — 국민연금·건강보험 (연금보험료공제 정확 산정)
- `severance-pay` — 퇴직소득세 (분류과세, 연말정산 합산 안 함)
